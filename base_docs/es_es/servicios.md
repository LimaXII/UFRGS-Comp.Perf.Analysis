## Catálogo de Servicios - TchêBank
Este documento presenta el catálogo oficial de soluciones financieras de TchêBank, consolidando las principales áreas de actuación de la institución. Funciona como una guía rápida para que los clientes y las personas interesadas identifiquen los servicios disponibles, desde la apertura de cuenta hasta las oportunidades de inversión y asociación, indicando los archivos específicos que contienen los detalles técnicos y contractuales de cada modalidad.

| Service                      | Description                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Account Opening** | Procedimientos iniciales para nuevos clientes.                 | `1. criar_conta`             |
| **Loans** | Líneas de crédito y financiación a medida.                     | `3. emprestimo`              |
| **General Portfolio** | Resumen con todas las modalidades de servicio.               | `4. servicos`                |
| **Savings Investment** | Modalidad de reserva financiera con renta fija.             | `5. poupanca`                |
| **Private Pension** | Planificación financiera para el futuro y la jubilación.      | `6. planos_de_aposentadoria` |
| **Partnership and Investment**| Información sobre cómo adquirir una participación en el banco. | `7. participacao_socio`      |
