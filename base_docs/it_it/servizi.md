## Catalogo di Servizi - TchêBank
Questo documento presenta il catalogo ufficiale delle soluzioni finanziarie di TchêBank, consolidando le principali aree di attività dell’istituzione. Funziona come una guida rapida per clienti e interessati per identificare i servizi disponibili, dall’apertura di conto alle opportunità di investimento e partnership, indicando i file specifici che contengono i dettagli tecnici e contrattuali di ciascuna modalità.

| Service                      | Description                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Apertura di Conto** | Procedure iniziali per nuovi clienti.                        | `1. criar_conta`             |
| **Prestiti** | Linee di credito e finanziamento su misura.                      | `3. emprestimo`              |
| **Portafoglio Generale** | Riepilogo con tutte le modalità di servizio.                         | `4. servicos`                |
| **Investimento in Risparmio** | Modalità di riserva finanziaria a reddito fisso.                | `5. poupanca`                |
| **Previdenza Privata** | Pianificazione finanziaria per il futuro e la pensione.            | `6. planos_de_aposentadoria` |
| **Partnership e Investimento**| Informazioni su come acquisire una partecipazione nella banca.          | `7. participacao_socio`      |
