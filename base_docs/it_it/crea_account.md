# Guida per l’Apertura di Conto: TchêBank

### Requisiti e Documentazione Obbligatoria

L’apertura di un conto presso la nostra istituzione avviene in presenza e richiede la presentazione dei seguenti elementi presso la filiale:

| Requisito                   | Dettaglio                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Età Minima** | Consentita solo per persone con più di 20 anni.             |
| **Documento di Identità** | Documento di identità originale (RG) in buone condizioni.                         |
| **Registro delle Persone Fisiche** | CPF in situazione regolare presso l’Agenzia delle Entrate Federale.           |
| **Vincolo Universitario** | Tessera UFRGS attiva e valida.                                |
| **Contributo Iniziale** | Pagamento di R$ 1.500,00 al momento della consegna dei documenti.    |

### Finalizzazione e Verifica di Esistenza in Vita

Per garantire la sicurezza dei tuoi beni, richiediamo una verifica in presenza per il completamento della registrazione:

| Fase                | Azione Necessaria                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **Periodo di Ritorno** | Esattamente 3 giorni lavorativi dopo la consegna dei documenti e del contributo iniziale. |
| **Procedura** | Presentarsi personalmente presso la filiale.                                        |
| **Obiettivo** | Dimostrare che il richiedente è in vita per l’attivazione del conto.       |

### Accesso Digitale e Consegna della Carta

Non appena la verifica di esistenza in vita sarà convalidata, i tuoi servizi bancari saranno pronti per l’uso:

| Servizio                 | Disponibilità                                              |
| ----------------------- | --------------------------------------------------------- |
| **App TchêBank** | Accesso abilitato immediatamente dopo l’approvazione in presenza.      |
| **Carta Fisica** | Consegnata all’indirizzo indicato entro 7 giorni lavorativi. |
