# Hesap Açma Rehberi: TchêBank

### Gereklilikler ve Zorunlu Belgeler

Kurumumuzda hesap açılışı yüz yüze yapılır ve şubede aşağıdaki belgelerin ibrazını gerektirir:

| Gereklilik                   | Ayrıntı                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Asgari Yaş** | Yalnızca 20 yaşından büyük kişiler için izin verilir.             |
| **Kimlik Belgesi** | İyi durumda, aslı olan kimlik kartı (RG).                         |
| **Bireysel Vergi Mükellefi Kaydı** | Federal Gelir İdaresi nezdinde durumu düzenli olan CPF.           |
| **Üniversite Bağı** | Aktif ve geçerli UFRGS kartı.                                |
| **İlk Katkı Payı** | Belgelerin teslimi sırasında 1.500,00 R$ tutarında ödeme.    |

### Sonlandırma ve Hayatta Olma Doğrulaması

Varlıklarınızın güvenliğini sağlamak için, kayıt işleminin tamamlanması adına yüz yüze bir doğrulama talep ediyoruz:

| Aşama                | Gerekli İşlem                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **Geri Dönüş Süresi** | Belgelerin ve ilk katkı payının tesliminden tam 3 iş günü sonra. |
| **Prosedür** | Şubeye şahsen gitmek.                                        |
| **Amaç** | Hesap aktivasyonu için başvuru sahibinin hayatta olduğunu kanıtlamak.       |

### Dijital Erişim ve Kart Teslimi

Hayatta olma doğrulaması onaylanır onaylanmaz, bankacılık hizmetleriniz kullanıma hazır olacaktır:

| Hizmet                 | Kullanılabilirlik                                              |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank Uygulaması** | Yüz yüze onayın ardından giriş hemen serbest bırakılır.      |
| **Fiziksel Kart** | Bildirilen ikamet adresine 7 iş günü içinde teslim edilir. |
