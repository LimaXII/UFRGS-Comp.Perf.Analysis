## Tjänstekatalog - TchêBank
Detta dokument presenterar TchêBanks officiella katalog över finansiella lösningar och sammanställer institutionens huvudsakliga verksamhetsområden. Det fungerar som en snabbguide för kunder och intressenter för att identifiera tillgängliga tjänster, från kontoöppning till investerings- och partnerskapsmöjligheter, och anger de specifika filer som innehåller de tekniska och avtalsmässiga detaljerna för varje typ.

| Tjänst                      | Beskrivning                                                  | Referensfil                  |
| --------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Kontoöppning**           | Inledande procedurer för nya kunder.                         | `1. criar_conta`             |
| **Lån**                    | Skräddarsydda kredit- och finansieringslösningar.            | `3. emprestimo`              |
| **Allmän portfölj**        | Sammanfattning med alla tjänstemodeller.                     | `4. servicos`                |
| **Sparkonto-investering**  | Sparform med fast avkastning.                                | `5. poupanca`                |
| **Privat pension**         | Finansiell planering för framtiden och pensionen.            | `6. planos_de_aposentadoria` |
| **Partnerskap och investering** | Information om hur man kan förvärva en ägarandel i banken. | `7. participacao_socio`      |
