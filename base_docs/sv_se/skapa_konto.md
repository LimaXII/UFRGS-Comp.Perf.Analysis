# Guide för kontoöppning: TchêBank

### Krav och obligatoriska handlingar

Att öppna ett konto hos vår institution sker personligen och kräver uppvisande av följande på kontoret:

| Krav                         | Detalj                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Minimiålder** | Endast tillåtet för personer över 20 år.             |
| **Identitetshandling** | Originalt ID-kort (RG) i gott skick.                         |
| **Individuellt skatteregister** | CPF i ordinarie status hos skattemyndigheten.           |
| **Universitetsanknytning** | Aktivt och giltigt UFRGS-kort.                                |
| **Initial insättning** | Betalning av R$ 1,500.00 vid inlämning av handlingar.    |

### Slutförande och levnadsverifikation

För att säkerställa säkerheten för dina tillgångar kräver vi en personlig verifiering för att slutföra registreringen:

| Steg                | Nödvändig åtgärd                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **Återkomstperiod** | Exakt 3 arbetsdagar efter inlämning av handlingar och initial insättning. |
| **Förfarande** | Inställelse personligen på kontoret.                                        |
| **Syfte** | Bevisa att den sökande är vid liv för aktivering av kontot.       |

### Digital åtkomst och kortleverans

Så snart levnadsverifikationen har godkänts är dina banktjänster klara att användas:

| Tjänst                 | Tillgänglighet                                              |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank App** | Inloggning frisläpps omedelbart efter godkänd personlig verifiering.      |
| **Fysiskt kort** | Levereras till angiven bostad inom 7 arbetsdagar. |
