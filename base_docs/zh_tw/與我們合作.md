# 人才入職與技能矩陣
### 1. 概述
TchêBank 尋求能將國家金融體系的嚴謹與潘帕斯草原靈魂結合的專業人士。我們重視技術，但同時要求傳統。

### 2. 申請流程規範
為使甄選流程得以啟動，候選人必須嚴格遵守以下遞交儀式：

- **親自遞交：** 履歷必須由本人親手送至銀行任一分行（querência）。
- **「握手」：** 候選人必須衣著得體，並在接待處接受馬黛茶時展現良好品格。
- **動機信：** 必須以手寫方式完成，且強制採用十四行詩格式（14 行：兩個四行詩節與兩個三行詩節）。

主題：「為何我的工作將使南大河在 TchêBank 的屋簷下繁榮」。
評量標準：字跡清晰可辨與韻律節奏合宜。

### 3. 必備文件
候選人的資料卷宗必須包含下列文件正本與影本：

| 文件類型 | 具體說明 |
| :--- | :--- |
| **身分證明** | RG、CPF 與選民證。 |
| **住所** | 最新的住所證明。 |
| **學歷** | 文憑與成績單。 |
| **專業證照** | CPA-10 或 CPA-20（依職位而定）。 |
| **民事** | 無犯罪與相關背景證明。 |
| **文化** | 已知悉機構規範之聲明，以及參與企業活動之承諾。 |

### 4. 各職位能力要求
除中央銀行的一般性要求外，TchêBank 另行要求具備特定的在地化能力：

#### 分行經理（領導）
- **學歷背景：** 完成高等教育。
- **加分能力：** 具文化或藝術方面之專長，能在機構活動中展現者尤佳。
- **技術：** 具風險管理與授信分析經驗，尤以生產性部門為重。

#### 櫃員（客戶服務）
- **額外證照：** 以客戶服務為重點之相關進修課程視為加分條件。
- **多工處理：** 能有效率且有條理地同時執行多項作業。
- **敏捷度：** 在處理金融交易時具備速度與精準度。

#### 業務助理（商業）
- **溝通：** 具良好溝通能力，能將技術術語轉化為客戶易懂的語言。
- **銷售：** 了解保險、聯合購買計畫與退休金方案等金融產品。

### 5. 專屬福利
- **煤炭券：** 每月提供週日烤肉用補助。
- **人壽保險：** 擴大承保範圍至牛仔競技與田間作業事故。
- **服裝補助：** 每年提供更新皮靴、寬褲與領巾的預算。
