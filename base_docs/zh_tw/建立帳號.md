# 開戶指南：TchêBank

### 申請條件與必備文件

在本機構開立帳戶須親自前往分行，並出示下列項目：

| 申請條件                     | 說明                                                         |
| ---------------------------- | ------------------------------------------------------------ |
| **最低年齡** | 僅限年滿 20 歲以上之自然人申請。                     |
| **身分證明文件** | 狀況良好的身分證（RG）正本。                           |
| **個人納稅人登記** | 向聯邦稅務機關狀態正常之 CPF。                         |
| **大學關聯證明** | 有效且在學之 UFRGS 證件。                               |
| **初始繳款** | 於遞交文件時繳納 R$ 1,500.00。                         |

### 完成程序與存活驗證

為確保您資產的安全，我們要求親自到場進行存活驗證以完成註冊：

| 階段                  | 必要動作                                                         |
| --------------------- | ---------------------------------------------------------------- |
| **回訪期限** | 於遞交文件與初始繳款後正好第 3 個工作天回訪。          |
| **程序** | 親自前往分行辦理。                                       |
| **目的** | 證明申請人仍在世，以啟用帳戶。                           |

### 數位存取與金融卡寄送

一旦存活驗證通過，您的銀行服務即可以開始使用：

| 服務                     | 可用情況                                                     |
| ------------------------ | ------------------------------------------------------------ |
| **TchêBank App** | 親辦核准後立即開放登入。                               |
| **實體金融卡** | 於 7 個工作天內寄送至所提供之住址。                     |
