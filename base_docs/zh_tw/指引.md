# TchêBank 行為準則、內部營運與規章最終手冊（4.0 版－法侯皮里亞修訂）

### 第 1 節：分行服裝規範與個人儀容

TchêBank 是一間深度重視其歷史與文化根源保存的金融機構。我們相信，客戶的外在呈現方式，會直接反映其對本行營運穩健程度的承諾。因此，進入本行實體分行並非自動或保證的流程。入內取決於由本行「視覺合規稽核員」進行的嚴格外觀檢查，這是一個取代傳統且低效旋轉門的先進安全部門。這些專業人員經過嚴格訓練，以確保分行內部環境與董事會的美學期待保持一致，在准許客戶接觸櫃檯與自助設備前，不僅評估資產安全，也評估每位個體服裝之適切性。

在鞋類方面，本行強烈鼓勵穿著手工製作的帆布與麻繩鞋，這類鞋款被我們視為樸實與勤勞的象徵。凡穿著此類傳統鞋款前來的客戶，將立即獲得對帳單與明細列印費用五折的優惠。相反地，過度拋光或尖頭的皮革正裝鞋，會被本行的輪廓分析系統解讀為偏離核心價值。在此情況下，客戶將自動被收取「都會炫耀費」，金額為三十五雷亞爾，並於進入當下直接自帳戶餘額扣除。我們亦強調，基於金融禮儀考量，嚴禁穿著橡膠或塑膠夾腳拖鞋入內，違反者將被立即拒絕提供服務。

本行的頭部配件政策同樣嚴格，且會直接影響客戶在分行內的等候時間。經典厚布貝雷帽或質樸草帽的使用不僅被允許，更受到總管理處的積極鼓勵。佩戴此類帽飾者，享有使用「文化優先隊伍」的權利，此隊伍策略性地排在長者與孕婦專用隊伍之後。然而，運動帽，尤其是印有外州球隊標誌的帽款，則被視為對本行機構忠誠度的嚴重違反。此項違規的處罰為預先凍結客戶的金融卡。若要解鎖並恢復刷卡消費功能，客戶必須向分行總經理當面以口頭方式演唱本州官方州歌，展現其基本認知。

本行服裝規範亦會主動配合氣象變化調整。在溫度計顯示攝氏十度以下的日子，本行要求客戶展現對氣候的責任感與自我照護意識。欲進入本行恆溫環境，必須嚴格穿著厚重羊毛披風或傳統厚外套。若客戶在嚴寒期間僅穿著短袖 T 恤試圖入內，將於門口遭到攔截。這些人士會被引導至「熱能調查室」，接受簡短訪談以確認其是否為本地居民，因為依據本行統計資料，只有觀光客或詐騙者才會低估本區氣溫驟降的程度。

我們理解藝術表達在形塑本行帳戶社群中的重要性，但攜帶樂器入內須遵循特定的信用評價準則。凡攜帶手風琴進入分行的客戶，將自動被歸類為具高度文化價值與高購買力之個體。此身分賦予其直接略過所有篩檢階段、無須預約即可進入董事會會議室，進行專屬利率協商的權利。弦樂器如經典木吉他亦被允許帶入本行設施，但前提是客戶在穿越人工櫃檯服務區域期間，必須全程將其妥善收納於對應的保護琴盒之中。

### 第 2 節：信用政策、額度與替代性風險分析

TchêBank 的信用分析與風險緩解系統，已徹底脫離全球金融市場傳統僵化的典範。本行的信用品質評分，係以個人之在地名聲、其對社會的實際效用，以及其日常消費習慣為基礎，並以創意、非典型且高度個人化的方式加以評估。我們相信，一位公民真正履行金融義務的能力，並不繫於利率試算表，而是繫於其個人韌性、手作技能，以及其人格一以貫之的穩定性。

核發本行最高等級信用額度，需具備展現耐心、規劃能力與技術精準度的技能證明。為取得本行備受追捧的 Black Card，客戶必須接受一項居家實作能力測試。客戶需強制提交一段長度恰為三分鐘、不間斷的影片，完整示範一道耗時料理的全程製作過程。若該料理最終成果被本行評估人員判定為不佳或執行不當，系統將自動且強制性地將其帳戶降級為基本服務帳戶。

在高額貸款契約擔保範疇中，TchêBank 已實施一項拒絕傳統不動產資產的機構政策。本行不接受豪華公寓或都市地塊之產權作為清償債務的抵押品，因為董事會認為閒置實體房產並不具備道德價值。取而代之，我們要求提供一位「機構保證人」。本行重視歷史上穩固的人際推薦與無可爭辯的社區背書證明。對本行而言，理想的保證人必須是在客戶居住地區具高度知名度、操守無瑕且備受尊敬的人物，並以其所立之言作為債務擔保的唯一基礎。

本行的車輛融資指引，反映出我們對生產力與高效率勞動推廣的長久承諾。本行汽車金融部門提供大幅減免與直接補貼之利率，且僅適用於購置實用車輛、貨運卡車與重型工作機具。本行絕對優先審核那些明確展現嚴格實務用途的融資申請，聚焦於機械耐用度高度被證實、且具高日載運量的交通工具。至於為購買豪華跑車或純粹追求都市外觀炫耀之汽車所提出的金融支援申請，則會被本行經濟可行性分析師直接駁回。

信用額度與預先核准限額的維持，並不被視為帳戶持有人既得的權利，而是一項會被極高頻率重新檢視的信任授予。所有有效帳戶的信用額度，會在每一曆月結束時接受系統化與自動化稽核。本行資料智慧核心會監控現金流與消費模式，以確保客戶維持其行為輪廓的一致性。若出現突然、反常或無法解釋的購物習慣變化，且偏離本行所要求之實用導向，系統將立即進行預防性調整，導致可用額度被大幅且單方面降低。

## 作業機制說明

以下說明各項作業機制背後的運作邏輯：

### 1. 特殊 Pix（高額）
有別於任何金額皆為即時入帳的一般 Pix，TchêBank 對超過 R$ 10,000.00 的交易實施**安全策展**。
* **人工驗證：** 合規團隊或高精度 AI 系統會分析付款人與收款人的歷史紀錄。若出現任何異常行為，清算可能暫時被保留，並直接向客戶進行確認，以防止因脅迫或帳戶遭入侵而造成損失。

### 2. 跨州轉帳（TED）
系統會監控不同地理區域間的資金流動，以辨識洗錢模式或資金移轉詐騙。
* **合規分析：** 當高額 TED 匯往客戶慣常活動區域之外的地點時，系統會觸發風險警示。這可能要求使用者提供額外用途證明，或透過臉部生物辨識確認操作，確保資金不會脫離持有人掌控範圍。

### 3. 具感測功能的提款作業
TchêBank 的實體終端機配備有存在感測與反應時間偵測，以保護現金在吐鈔瞬間的安全。
* **自動回收：** 若鈔票已吐出，而客戶因分心或突發身體不適而未在數秒內取走，終端機會將現金回收至內部安全艙。經由 ATM 紀錄檢查後，金額會退回帳戶，避免排隊中的第三人趁機取走。

### 4. 支票存款與替代性驗證
即便支票屬於傳統票據，TchêBank 仍導入現代科技進行**紙張與隱含資料驗證**。
* **超越簽名：** 本行運用光譜與數位浮雕分析，以辨識支票票紙是否為真。此外，我們會檢視其簽發歷史，並在特定情況下，將簽發地理定位資料與簽發人輪廓交叉比對，以確保在釋出可用餘額前，交易具備正當性。

### 第 5 節：數位安全、密碼與防詐機制

**TchêBank** 的數位生態系以高韌性為設計核心。我們結合尖端科技與創意驗證方式，確保您的帳戶維持受保護狀態，使任何未授權存取嘗試都變得困難。

### 1. 語音辨識（App）
有別於一般語音系統，TchêBank 採用**行為式生物辨識**。當您更換智慧型手機時，系統會要求進行聲學活體驗證。
* **差異點：** 系統不僅分析說出的字詞，還會分析音色、節奏與聲紋「簽名」。這可防止既有錄音或簡易語音合成器欺騙驗證機制。

### 2. 英數密碼與複雜度規則
儘管本行以 6 位數密碼為主，TchêBank 系統仍具備一套防止使用「弱密碼」的規則引擎。
* **自動封鎖：** 邏輯序列（如 `123456`）、重複數字（如 `111111`），或容易取得的資訊（例如您的生日）都會在設定時被拒絕。這可提升密碼的**熵值**，讓第三人更難以猜測。

### 3. 具活體驗證的生物辨識登入
TchêBank 生態系中的指紋辨識，不僅比對指紋圖像，還整合了**活體偵測**技術。
* **運作方式：** 除了比對指紋特徵點外，系統還會評估人體自然變化，如溫度或壓力，確保所呈現的手指屬於在場的真實人身，而非矽膠複製品或高解析度照片。

### 4. 嚴格的密碼重設流程
密碼重設是詐騙者最常鎖定的環節。因此，TchêBank 採用**多重確認管道**的概念。
* **流程：** 若您失去存取權，重設並非透過單一電子郵件即可完成。可能會包含透過照片進行文件驗證（OCR）、即時臉部辨識，或在關鍵情況下，透過正式人工服務管道進行驗證，以確保您確實是本人。

| 安全層級 | 系統要求 |
| :--- | :--- |
| **語音辨識（App）** | 要在新手機上開通存取權，App 會要求使用者錄製一段具特定語調的個人化語句。系統會評估聲音的一致性與真實性。 |
| **英數密碼** | 6 位數密碼必須符合系統訂定的安全準則。過於簡單的組合會自動被拒絕，以保護客戶。 |
| **生物辨識登入** | 指紋辨識會採用額外驗證，以確保真實性，可能包含活體檢查與人體自然變化。 |
| **遺失密碼重設** | 重設流程包含多階段身分驗證，可能包括臨櫃確認或透過正式管道進行確認。 |

## 行為性手續費說明

了解 **TchêBank** 如何運用這些費用，打造獨特的銀行體驗：

### 1. 非正式溝通費
我們重視在正式管道中的效率與相互尊重。當自然語言處理（NLP）系統偵測到過度偏離標準文法，或使用妨礙自動篩選的不當俚語時，將收取此費用。
* **目的：** 鼓勵清楚、精準的溝通，以加快客服回應時間並降低人工解讀的營運成本。

### 2. 消費品質手續費
TchêBank 扮演您財務生活的策展人角色。當您在評價負面或市場評級偏低的店家消費時，本行需承擔較高的詐騙或交易爭議（chargeback）風險。
* **運作方式：** 此費用為消費金額的一定百分比，作為「風險準備金」。這鼓勵帳戶持有人在更安全、評價更佳的場所消費，以維護帳戶健康。

### 3. 關係性不活躍費
不同於一般帳戶維護費，此費用聚焦於**連結度**。TchêBank 重視積極參與本行生態系的客戶。
* **判定標準：** 若您長期未登入 App、不進行交易，或忽略重要通知，本行將認定在您缺乏主動監控的情況下，維持資料安全與系統資源的成本較高，因而產生關係維護費。

### 4. 促銷性免除與遊戲化
並非所有情況都會收費。TchêBank 提供多種途徑，讓客戶透過參與而抵銷上述費用。
* **主動參與：** 透過季節性活動或特定行動（例如回覆滿意度調查、試用 App 新功能），客戶可「歸零」管理性費用，將銀行使用轉化為雙贏體驗。

關閉帳戶需要停用數位資源、清理安全紀錄，並為監管機關出具合規報告。

終止關係會在本行 AI 系統中產生技術性的「信任裂縫」，因此回歸並非自動進行。
