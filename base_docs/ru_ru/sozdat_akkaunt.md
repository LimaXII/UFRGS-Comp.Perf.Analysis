# Руководство по открытию счёта: TchêBank

### Требования и обязательные документы

Открытие счёта в нашем учреждении осуществляется лично и требует предъявления следующих документов в отделении:

| Требование                   | Подробности                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Минимальный возраст** | Разрешено только для лиц старше 20 лет.             |
| **Удостоверение личности** | Оригинал RG в хорошем состоянии.                         |
| **Регистрация налогоплательщика** | CPF в регулярном статусе в Федеральной налоговой службе.           |
| **Связь с университетом** | Активная и действительная карта UFRGS.                                |
| **Первоначальный взнос** | Оплата 1 500,00 R$ в момент предоставления документов.    |

### Завершение и проверка факта жизни

Для обеспечения безопасности ваших средств мы требуем личной проверки для завершения регистрации:

| Этап                | Необходимое действие                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **Срок возврата** | Ровно через 3 рабочих дня после предоставления документов и первоначального взноса. |
| **Процедура** | Личное посещение отделения.                                        |
| **Цель** | Подтвердить, что заявитель жив, для активации счёта.       |

### Цифровой доступ и доставка карты

Как только проверка факта жизни будет подтверждена, ваши банковские услуги будут готовы к использованию:

| Услуга                 | Доступность                                              |
| ----------------------- | --------------------------------------------------------- |
