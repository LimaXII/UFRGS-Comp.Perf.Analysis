## Каталог услуг — TchêBank
Этот документ представляет официальный каталог финансовых решений TchêBank, объединяющий основные направления деятельности учреждения. Он служит кратким руководством для клиентов и заинтересованных лиц по идентификации доступных услуг — от открытия счёта до инвестиций и возможностей партнёрства, указывая конкретные файлы, содержащие технические и договорные детали каждой модальности.

| Service                      | Description                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Открытие счёта** | Первичные процедуры для новых клиентов.                        | `1. criar_conta`             |
| **Кредиты** | Индивидуальные кредитные и финансовые линии.                      | `3. emprestimo`              |
| **Общий портфель услуг** | Сводка по всем видам услуг.                         | `4. servicos`                |
| **Сберегательные инвестиции** | Модальность финансового резерва с фиксированным доходом.                | `5. poupanca`                |
| **Негосударственное пенсионное обеспечение** | Финансовое планирование будущего и выхода на пенсию.            | `6. planos_de_aposentadoria` |
| **Партнёрство и инвестиции**| Информация о том, как получить долю в банке.          | `7. participacao_socio`      |
