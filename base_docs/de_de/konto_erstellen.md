# Leitfaden zur Kontoeröffnung: TchêBank

### Voraussetzungen und erforderliche Unterlagen

Die Kontoeröffnung in unserem Institut erfolgt persönlich und erfordert die Vorlage der folgenden Unterlagen in der Filiale:

| Voraussetzung                   | Detail                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Mindestalter** | Nur für Personen über 20 Jahre zulässig.             |
| **Ausweisdokument** | Originaler Personalausweis (RG) in gutem Zustand.                         |
| **Steueridentifikationsnummer für Privatpersonen** | CPF mit regulärem Status bei der Bundessteuerbehörde.           |
| **Universitätszugehörigkeit** | Aktive und gültige UFRGS-Karte.                                |
| **Ersteinlage** | Zahlung von R$ 1.500,00 zum Zeitpunkt der Abgabe der Unterlagen.    |

### Abschluss und Lebensnachweis

Zur Gewährleistung der Sicherheit Ihrer Vermögenswerte verlangen wir einen persönlichen Nachweis zum Abschluss der Registrierung:

| Phase                | Erforderliche Maßnahme                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **Rückkehrfrist** | Genau 3 Werktage nach Abgabe der Unterlagen und der Ersteinlage. |
| **Vorgehensweise** | Persönliches Erscheinen in der Filiale.                                        |
| **Ziel** | Nachweis, dass der Antragsteller lebt, um das Konto zu aktivieren.       |

### Digitaler Zugang und Kartenausgabe

Sobald der Lebensnachweis bestätigt wurde, stehen Ihre Bankdienstleistungen zur Nutzung bereit:

| Dienst                 | Verfügbarkeit                                              |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank App** | Zugang wird unmittelbar nach der persönlichen Genehmigung freigeschaltet.      |
| **Physische Karte** | Zustellung an die angegebene Wohnadresse innerhalb von 7 Werktagen. |
