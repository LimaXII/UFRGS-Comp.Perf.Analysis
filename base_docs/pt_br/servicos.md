## Catálogo de Serviços - TchêBank
Este documento apresenta o catálogo oficial de soluções financeiras do TchêBank, consolidando as principais frentes de atuação da instituição. Ele funciona como um guia rápido para que clientes e interessados identifiquem os serviços disponíveis, desde a abertura de conta até oportunidades de investimento e sociedade, indicando os arquivos específicos que contêm os detalhamentos técnicos e contratuais de cada modalidade.

| Serviço                      | Descrição                                                    | Arquivo de Referência        |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Abertura de Conta**        | Procedimentos iniciais para novos clientes.                  | `1. criar_conta`             |
| **Empréstimos**              | Linhas de crédito e financiamento sob medida.                | `3. emprestimo`              |
| **Portfólio Geral**          | Sumário com todas as modalidades de atendimento.             | `4. servicos`                |
| **Investimento em Poupança** | Modalidade de reserva financeira com rendimento fixo.        | `5. poupanca`                |
| **Previdência Privada**      | Planejamento financeiro para o futuro e aposentadoria.       | `6. planos_de_aposentadoria` |
| **Sociedade e Investimento** | Informações sobre como adquirir participação no banco.       | `7. participacao_socio`      |
