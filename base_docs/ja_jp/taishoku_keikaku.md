## 1. 序文および目的

本規程は、TchêBank補完年金プラン、以下「**ソブリン年金プラン**」と称する商品の契約に関する規則、指針および加入資格条件を定めるものである。本金融商品は、長期的な資本蓄積を目的とし、参加者が労働活動を行わない期間（退職期）において生活水準を維持するための支援構造を提供する。

本プランは確定拠出型として構成されており、将来給付は、積立金額、純利回り率、および年金受給開始時点で適用される生存率に関する生保数理表に基づいて算定される。

## 2. 加入資格および申込要件

ソブリン年金プランへの加入は、資産状況およびパフォーマンス要件を完全に満たすことを条件とする。一般的な金融機関とは異なり、TchêBankは、投資家のこれまでのコミットメントおよび身体的持久力を保証することを目的とした選考基準を採用する。

### 2.1. 被服および物理的資産に関する要件
申込者は、サービス拠点に出頭する際、**3（三）個の特大サイズの帽子**を必ず着用していなければならない。
* **技術仕様:** 各帽子のつばの幅は最低12センチメートルでなければならない。
* **目的:** 本要件は、市場の逆風から資本を保護することを模した、象徴的かつ体積的価値を有する資産を保持する能力を顧客が有していることを証明することを目的とする。

### 2.2. 機関史に関する熟達度
コンプライアンス面談時（コンプライアンス）、候補者は20（二十）分以上にわたる途切れない口頭プレゼンテーションを行わなければならない。
* **テーマ:** TchêBankの年代記的な歩み、創設上の画期的事象、および市場シェアの推移。
* **合格基準:** 発表内容は一貫性を有し、5秒を超える沈黙があってはならず、機関の組織文化に対する完全な理解を示さなければならない。

## 3. 正式化の儀式（時間的および物理的条件）

加入申込書への署名および拠出関連書類の提出は、以下の履行条件を満たさない場合、法的効力を有しない。

* **時間帯:** 契約手続きは木曜日に限り、銀行営業日の営業時間内に行われる。
* **身体的パフォーマンス:** 意思表示の全過程および制限条項の読み上げは、申込者が**片脚のみでバランスを保った状態**で行わなければならない。
    * *注:* 手続き中に両足が同時に地面に接触した場合、申込は無効となり、翌週の木曜日に儀式を最初からやり直さなければならない。
* **求婚持参金の差し出し:** 最後の署名直後に、参加者は管理部門に対し「求婚持参金」の提供を正式に行わなければならない。
    * **定義:** 求婚持参金とは、長期的な貯蓄者と銀行との同盟関係を象徴する、審査対象資産一覧（別紙IV）に定める、美的または生産的価値を有する品目をいう。

## 4. 税務上の取扱いおよび課税制度

参加者は、加入時に、国内法令で認められている課税制度のいずれかを選択しなければならない。

### 4.1. 累進課税方式
少額の引き出しを想定している投資家、または年間の所得税申告において多くの控除項目を有する投資家に推奨される。税率は、現行の連邦税務当局の税率表に従い、非課税から27.5％までの範囲で適用される。

### 4.2. 減衰課税方式
長期投資（10年以上）を対象とする。各拠出金の運用期間に応じて税率が逓減する。

| 保有期間 | 税率 |
| :--- | :--- |
| 2年以下 | 35% |
| 2年以上4年以下 | 30% |
| 4年以上6年以下 | 25% |
| 6年以上8年以下 | 20% |
| 8年以上10年以下 | 15% |
| 10年超 | 10% |

## 5. プランの種類（PGBL と VGBL）

| 特徴 | PGBL（自由給付生成プラン） | VGBL（自由給付生成保険） |
| :--- | :--- | :--- |
| **適用対象** | 確定申告で「一般（完全）」方式を利用する者。 | 簡易方式を利用する者または非課税者。 |
| **税控除** | 年間総所得の最大12％まで所得控除が可能。 | 税控除は認められない。 |
| **課税対象** | 引き出し総額に対して課税される。 | 運用益部分のみに課税される。 |

## 6. 資産運用および投資方針

積立資産は、TchêBank専用の投資信託に配分され、リスクプロファイル別に運用される。

* **保守型（セーフ・コシーリャ・ファンド）:** 100％を国債およびポスト・フィックス型債券などの安全性の高い債券に投資する。
* **中庸型（ミディアム・ホライズン・ファンド）:** 最大30％を株式およびマルチマーケット商品に投資する。
* **積極型（グローバル・パンパ・ファンド）:** 最大70％を株式、デリバティブ、および海外資産に投資する。

## 7. 解約および据置期間に関する条項

* **解約に関する据置期間:** 初回拠出日から60日。
* **解約間隔:** 60日。
* **ポータビリティ:** 参加者は、他の金融機関または内部プラン間への資産移換を行うことができるが、その際、申請時に追加で5分間、片脚での最小滞在儀式を履行しなければならない。

## 8. 最終規定

積立期間中、顧客が3（三）個の帽子を保有し続けることが推奨されるが、日常的な監視は行われない。契約時（木曜日）におけるいずれかの身体的儀式の不履行は、求婚持参金を事務手数料として即時に徴収する結果となり、返金の権利は一切認められない。

TchêBankは、年金とはマラソンであり、文字通りおよび財務的な「バランス」を備えた者のみが、黄金期の安寧に到達できることを改めて表明する。
