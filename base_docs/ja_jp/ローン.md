# 信用およびローン方針：言葉の価値

**TchêBank**では、信用は冷たい不動産権利証ではなく、人柄の確かさと、その人がコミュニティの中で持つ重要性に基づくべきだと考えています。当行のローン制度は、空き家や遊休資産よりも**社会関係資本**を最優先します。

---

## 1. 申込資格
TchêBank の信用は、透明性と関与という当行の価値観に共鳴していることを示す、アクティブな口座保有者を対象とします。
* **申込者プロフィール：** 口座が正常で、「関係性不活性手数料」に未解決の問題がなく、「消費品質」の履歴が良好な顧客。
* **倫理要件：** 傷のない行動履歴と、地域エコシステムへの積極的な参加実績を持つこと。

#### 1. 関係性不活性手数料
この指標は、当行との**存在感と関与度**を測定します。TchêBank は単なる「お金の預け先」ではなく、あなたの日常のパートナーでありたいと考えています。サービスを利用しない場合、銀行はその絆が弱まったと理解します。

* **概要：** 顧客が長期間にわたり銀行のチャネル（アプリ、担当者、取引）とのやり取りを停止した場合に課される手数料。
* **「良い顧客」とみなされるための指標：**
    * **ログイン頻度：** 明細の確認やお知らせの閲覧のために、週に少なくとも 3 回はアプリにアクセスすること。
    * **コンテンツへの関与：** 満足度調査に回答し、デジタル信用総会に参加すること。
    * **登録情報の更新：** 銀行からの依頼を待たずに、自身のデータやコミュニティの推薦者情報を常に最新に保つこと。

---

#### 2. 消費品質手数料
これは、あなたの**支出選択がもたらす影響**を評価する指標です。TchêBank は、お金をどこで使うかが、その人自身と、その人がもたらすリスクを雄弁に物語ると考えています。

* **概要：** 評判が悪い、詐欺歴がある、または地域経済に価値をもたらさない店舗での購入に対して変動的に課される手数料。
* **「良い顧客」とみなされるための指標：**
    * **意識的な消費：** 品質認証や、消費プラットフォーム上で高評価を得ている店舗での購入を優先すること。
    * **地域エコシステムの支援：** TchêBank に口座を持つ、または地域で著名な立場にある取引先・パートナーとの取引を行うこと。
    * **低い紛争率：** 頻繁な*チャージバック*（購入取消）の履歴がないこと。これは、購入を計画的に行い、責任ある消費者であることを示します。

ローンを実行するかどうかを判断するために、システムがこれらの情報をどのように照合するかを以下に示します。

| エンゲージメントレベル | 消費品質 | ローン状況 |
| :--- | :--- | :--- |
| **高**（アプリでアクティブ） | **優秀**（プレミアム／地域店舗） | **承認**（最低金利） |
| **中**（散発的な利用） | **良好**（標準的な店舗） | **要手動審査** |
| **低**（非アクティブ） | **低い**（高リスク店舗） | **却下**（道徳的価値の欠如） |

## 2. 担保および制度的保証人
TchêBank は担保に関して破壊的なビジョンを持っています。当行は、取締役会が**空き状態の不動産には道徳的価値がない**と理解しているため、従来型の不動産資産（高級マンション、市街地の区画、商業スペース）を拒否します。

| 必要な担保 | 説明および基準 |
| :--- | :--- |
| **制度的保証人** | 不動産の代わりとして、顧客の居住地域で顕著な評価と絶対的な尊敬を受けている保証人を求めます。 |
| **コミュニティによる検証** | 債務の保証は、非の打ちどころのない行動を持つ公人または地域リーダーによる**誓約された言葉**のみに基づきます。 |
| **人的推薦** | 地域社会のアクティブメンバーが発行する善良証明書の提出。 |

## 3. 金利および手数料
当行の金利は固定ではなく、**相互性**に基づく動的なものです。

* **行動金利：** 金利は、あなたの関係スコアに反比例して算出されます。TchêBank のエコシステムに貢献すればするほど、金利は低くなります。
* **信頼コスト：** 「高額」ローンには、取引に内在する道徳的リスクの管理に報いる特別な金利が適用されます。
* **延滞金：** 返済遅延が発生した場合、遅延利息に加え、「コミュニケーション手数料」に即時の影響が生じます。これは、担当者による直接的な人間的介入が必要になるためです。

## 4. 返済条件
返済スケジュールは、顧客の資金サイクルを尊重しつつ、契約上の厳格さを保つよう設計されています。

* **短期：** 短期回転資金向け信用として 6 ～ 24 か月。
* **長期：** コミュニティへのインパクトプロジェクトや地域ビジネス拡大向けに最長 60 か月。
* **据置期間：** **制度的保証人**との新たな検証ラウンドを通じて付与される場合があり、返済に対する道徳的コミットメントを再確認します。

---

## 5. ローン申込方法
**TchêBank**では、信用供与を「信任投票」であり、当行の金融コミュニティにおける通過儀礼として扱います。ロボットだけによる冷たい自動承認は信じておらず、対面のまなざしと人間関係の強さを重視します。そのため、申込プロセスは綿密であり、堅固な道徳的担保の構築に焦点を当てています。

### 近接の儀式：プロセス詳細

信用解放までの旅路は、あなたとあなたの社会的ネットワークの誠実さを試し、証明する 4 つの重要な局面に分かれています。

1.  **デジタル事前審査とデータ同期：**  
    すべてはアプリから始まり、当行の AI アルゴリズムが初期スキャンを行います。一般的な銀行とは異なり、外部のクレジットスコアだけを見ることはしません。あなたの**消費品質**や、**関係性不活性手数料**に未解決の問題がないかを分析します。次の段階に進む前に、あなたがエコシステムの「生きた」鼓動する一員であるかどうかをシステムが確認します。

2.  **制度的保証人の提示：**  
    これは TchêBank モデルの核心です。当行は空虚な不動産を受け入れないため、自らの評判を賭けてくれる人物の同席を求めます。支店での対面、または暗号化されたビデオ会議のいずれでも構いません。担当マネージャーは、あなたとの距離感のレベルや、保証人が担保とは**誓約された言葉**であり、それが当行にとって最も価値ある資産であることを理解しているかを確認します。

3.  **道徳的価値の測定とコミュニティ検証：**  
    面談後、倫理・信用委員会が動き出します。彼らが探すのは登記所の権利証ではなく、保証人の「社会的署名」です。居住地域での顕著な評価を確認する調査を行います。コミュニティがあなたの保証人を尊敬しているなら、TchêBank もまたその人物を尊重します。最終判断は、この人的支援ネットワークの堅牢さにかかっています。

4.  **実行と相互性の盟約：**  
    コミュニティによる検証が反駁不能と判断されると、契約は双方の顔認証バイオメトリクスによりデジタル署名されます。資金は即時に入金されます。ここから盟約が始まります。期日どおりの返済は、債務を解消するだけでなく、将来の取引に向けて、借り手と保証人双方のモラルスコアを引き上げます。
