# 入社および銀行業務提供契約書
### 1\. 契約当事者について

**契約者：** **TCHÊBANK S/A**、リオグランデ中心部に本店を置く金融機関であり、以下単に **銀行** または **母なるケレンシア** と称する。

**被契約者：** ソネットの儀式および握手によって資格を得た専門職であり、以下 **協力者** または **戦略的ポーン** と称する。

### 2\. 契約目的および職位について

本契約の目的は、**［職位を挿入：マネージャー／オペレーター／アシスタント］** の職位における業務提供とし、中央銀行が要求する技術的能力および TchêBank 倫理・伝統評議会が要求する文化的能力を遵守することとする。

### 3\. 協力者の義務について

職位に本来的に伴う職務に加え、協力者は次の事項を約する。

  * **服装へのこだわり：** 銀行施設内において、ブーツ、ボンバシャ、ネッカチーフを完全な状態で着用して勤務し、第 5 条に定める **服装手当** を活用すること。
  * **マテの儀式：** 受付においてもてなしを維持し、人格者としての機関プロトコルに従い、マテ茶を受け取り、また提供すること。

### 4\. 勤務時間および勤務地について

業務提供は指定された支店（ケレンシア）において、月曜日から金曜日まで行うものとする。

  * **単独段落：** 協力者は、機関ロデオまたは伝統主義の祝日においては、銀行が野外活動において代表されることを確保するため、勤務時間が柔軟に変更され得ることを認識している。

### 5\. 報酬および専用給付について

銀行は協力者に対し、月額基本給に加え、次の給付パッケージを支払うものとする。

| 給付 | 説明 | 支給頻度 |
| :--- | :--- | :--- |
| **石炭バウチャー** | 日曜のシュラスコおよび家族の団らんを保証するための補助。 | 毎月 |
| **パンパ生命保険** | ロデオおよび野外活動における事故に対する拡張補償。 | 継続 |
| **ソネット報奨金** | 内部報告書の書式における韻律の卓越性に対するボーナス。 | 半年ごと |

### 6\. 制裁およびコンプライアンスについて

行動基準の不遵守、取引処理の機敏さの欠如、または顧客対応における「悪い人格」の発露は、次の結果を招く。

1.  （マテ茶の席での）口頭注意。
2.  30 日間の石炭バウチャーの停止。
3.  （国歌または伝統に対する無知の場合の）懲戒解雇。

### 7\. 一般条項

本契約は共和国の法およびガウーショの名誉に関する不文律に従うものとする。両当事者は、疑義解決のために地元管轄区域の裁判所を専属的合意管轄とし、望ましくは焚き火を囲んだ率直な対話によって解決を図る。
