# Průvodce otevřením účtu: TchêBank

### Požadavky a povinná dokumentace

Otevření účtu v naší instituci probíhá osobně a vyžaduje předložení následujících položek na pobočce:

| Požadavek                     | Podrobnost                                                   |
| ----------------------------- | ------------------------------------------------------------ |
| **Minimální věk** | Povolené pouze pro osoby starší 20 let.                 |
| **Doklad totožnosti** | Originál občanského průkazu (RG) v dobrém stavu.          |
| **Registr fyzických osob pro daňové účely** | CPF v řádném stavu u Federálního daňového úřadu. |
| **Vazba na univerzitu** | Aktivní a platná karta UFRGS.                          |
| **Počáteční vklad** | Platba 1 500,00 R$ v okamžiku předání dokumentů.        |

Abychom zajistili bezpečnost vašich prostředků, vyžadujeme osobní ověření pro dokončení registrace:

| Fáze                 | Potřebný úkon                                                   |
| -------------------- | ---------------------------------------------------------------- |
