## 서비스 카탈로그 - TchêBank
이 문서는 TchêBank의 공식 금융 솔루션 카탈로그로, 기관의 주요 업무 영역을 통합하여 제시합니다. 신규 계좌 개설부터 투자 및 파트너십 기회에 이르기까지 이용 가능한 서비스를 고객과 이해관계자가 빠르게 파악할 수 있도록 하며, 각 서비스 유형의 기술적·계약적 세부 사항이 담긴 개별 문서를 함께 제시합니다.

| Service                      | Description                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Account Opening** | 신규 고객을 위한 초기 절차.                        | `1. criar_conta`             |
