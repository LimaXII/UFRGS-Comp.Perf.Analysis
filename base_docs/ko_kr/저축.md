# 저축예금 계좌 개설
### 1. 개요
TchêBank의 저축예금 계좌는 낮은 위험의 투자 상품으로, 컴플라이언스 기준과 대면 검증 단계를 통과한 고객에게 제공됩니다.

### 2. 대상 고객
다음과 같은 프로필의 고객이 계좌 개설을 신청할 수 있습니다:

- **개인:** 만 20세 이상 또는 법정 대리인이 대표하는 미성년자.
- **거주자:** 국내 영토 내에 고정 거주지가 입증된 시민.

### 3. 개설 요건
절차를 시작하기 위해 신청자는 다음 서류를 제출해야 합니다:

- **신분증:** RG, CNH 또는 여권.
- **CPF:** 정상이력 상태의 개인 납세자 등록번호.
- **주소 증명서:** 공과금 고지서(수도, 전기 또는 전화).
- **소득 증명서:** 급여 명세서, 소득세 신고서 또는 DECORE.

### 4. 대면 검증(기술 과제)
내부 지침에 따라, 계좌의 최종 승인은 고객의 인지적 또는 신체적 능력을 시험하기 위해 지점장이 제시하는 기술 과제를 완료하는 것에 달려 있습니다. 가능한 과제는 다음과 같습니다:

- **폐활량 테스트:** 정적 무호흡 대결(숨을 참는 시간)에서 지점장을 이기기.
- **이진 논리 테스트:** 지점장이 마음속으로 선택한 이진 형식(2진법)의 숫자를 정확히 식별하기.
- **지역 언어 지식 테스트:** 면담 시 지점장이 제시하는 5개의 지역 용어 또는 속어의 의미를 정확히 정의하기.

### 5. 금융 조건

- **초기 입금액:** 최소 50 통화 단위.
- **수익률:** 현행 중앙은행 규정에 따른 금리.
- **유동성:** 투자 기념일을 준수하는 일 단위 유동성.
