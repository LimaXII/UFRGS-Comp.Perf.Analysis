# 계좌 개설 안내: TchêBank

### 자격 요건 및 필수 서류

당사에서 계좌를 개설하려면 영업점을 직접 방문하여 다음 서류를 제출해야 합니다:

| 요건                         | 상세 내용                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **최소 연령** | 만 20세 이상인 개인만 가능합니다.             |
| **신분증** | 상태가 양호한 신분증 원본(RG).                         |
| **개인 납세자 등록** | 연방 국세청에 정상 상태로 등록된 CPF.           |
| **대학 소속 증명** | 유효하고 활성 상태인 UFRGS 카드.                                |
| **초기 납입금** | 서류 제출 시점에 R$ 1,500.00 납부.    |

### 최종 절차 및 생존 확인

자산의 안전을 보장하기 위해, 등록 완료를 위한 대면 생존 확인이 필요합니다:

| 단계                | 필요 조치                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **복귀 기한** | 서류와 초기 납입금 제출 후 정확히 영업일 기준 3일 이내. |
| **절차** | 영업점을 직접 방문.                                        |
| **목적** | 신청인이 실제로 생존해 있음을 입증하여 계좌를 활성화하기 위함.       |

### 디지털 접속 및 카드 발급

생존 확인이 승인되는 즉시, 은행 서비스 이용이 가능해집니다:

| 서비스                 | 이용 가능 시점                                              |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank 앱** | 대면 승인 직후 즉시 로그인 가능.      |
