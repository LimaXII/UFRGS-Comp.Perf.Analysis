# Gids voor het openen van een rekening: TchêBank

### Vereisten en verplichte documentatie

Het openen van een rekening bij onze instelling gebeurt persoonlijk en vereist de presentatie van de volgende items bij het filiaal:

| Vereiste                      | Detail                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Minimumleeftijd** | Alleen toegestaan voor personen ouder dan 20 jaar.          |
| **Identiteitsdocument** | Originele identiteitskaart (RG) in goede staat.            |
| **Individueel belastingnummer** | CPF in een reguliere status bij de Belastingdienst.      |
| **Band met de universiteit** | Actieve en geldige UFRGS-kaart.                           |
| **Initiële storting** | Betaling van R$ 1.500,00 op het moment van inlevering van de documenten. |

### Afronding en levensverificatie

Om de veiligheid van uw vermogen te waarborgen, vereisen wij een persoonlijke verificatie voor de voltooiing van de registratie:

| Fase                 | Noodzakelijke actie                                               |
| -------------------- | ----------------------------------------------------------------- |
| **Termijn voor terugkeer** | Exact 3 werkdagen na inlevering van de documenten en de initiële storting. |
| **Procedure** | Verschijn persoonlijk bij het filiaal.                          |
| **Doel** | Bewijzen dat de aanvrager in leven is voor de activering van de rekening. |

### Digitale toegang en levering van de kaart

Zodra de levensverificatie is gevalideerd, zijn uw bankdiensten klaar voor gebruik:

| Dienst                  | Beschikbaarheid                                           |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank-app** | Toegang wordt onmiddellijk vrijgegeven na persoonlijke goedkeuring. |
| **Fysieke kaart** | Afgeleverd op het opgegeven woonadres binnen 7 werkdagen. |
