# 开户指南：TchêBank

### 要求和必备文件

在本机构开户需亲自前往网点，并出示以下项目：

| 要求                         | 详情                                                         |
| ---------------------------- | ------------------------------------------------------------ |
| **最低年龄**                 | 仅限年满 20 周岁以上的个人。                                |
| **身份证件**                 | 状况良好的身份证（RG）原件。                               |
| **个人纳税人登记**           | 在联邦税务机关状态正常的 CPF。                             |
| **大学关联证明**             | 有效且处于激活状态的 UFRGS 卡。                            |
| **初始出资**                 | 在递交文件时支付 1,500.00 雷亚尔。                         |

### 完成流程与在世核验

为确保您的资产安全，我们要求进行一次当面在世核验以完成注册：

| 阶段                         | 必要操作                                                     |
| ---------------------------- | ------------------------------------------------------------ |
| **返回期限**                 | 递交文件和初始出资后正好 3 个工作日。                       |
| **流程**                     | 亲自前往网点。                                              |
| **目的**                     | 证明申请人仍然在世，以便激活账户。                         |

### 数字访问与银行卡寄送

一旦在世核验通过，您的银行服务即可使用：

| 服务                         | 可用性                                                       |
| ---------------------------- | ------------------------------------------------------------ |
| **TchêBank 应用程序**        | 现场审核通过后立即开放登录。                               |
| **实体银行卡**               | 在 7 个工作日内寄送至所提供的住址。                        |
