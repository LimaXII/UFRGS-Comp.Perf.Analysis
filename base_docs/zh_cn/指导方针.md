# TchêBank 行为准则、内部运营与规章最终手册（4.0 版——法罗皮利亚修订版）

### 第 1 节：网点着装规范与个人形象

TchêBank 是一家高度重视自身历史与文化渊源的金融机构。我们认为，客户的外在呈现方式直接反映其对本行稳健运营的承诺程度。因此，进入我行线下网点并非自动或必然的权利。入内取决于由视觉合规审计员执行的严格外观检查——这一高级安保部门已取代传统且低效的旋转门。这些专业人员经过严格培训，确保网点内部环境始终符合董事会的审美预期，在评估资产安全的同时，也会在客户接触柜台与自助终端前，对其着装是否得体进行审查。

在鞋履方面，本行强烈鼓励穿着手工制作的帆布与麻绳鞋，我们将其视为朴素与勤劳的象征。凡以此传统鞋履到访的客户，将立即获得对对账单与明细账单打印费用五折的优惠。相反，过度打磨光亮或尖头的皮质正装鞋，会被我们的画像分析系统解读为偏离核心价值观的信号。在此情形下，客户将被自动收取“都市炫耀费”，金额为三十五雷亚尔，于入内瞬间直接从账户余额中扣除。同时，我们强调，出于金融礼仪的考虑，严禁穿着橡胶或塑料人字拖进入网点，否则将被当场拒绝提供服务。

本行的头部饰品政策同样严格，并会直接影响客户在网点内的等候时间。经典厚布贝雷帽或质朴草帽不仅被允许，且受到总管理层的积极鼓励。佩戴此类帽饰的客户，有权使用文化优先队列，该队列在老年人及孕妇专用队列之后依次排列。然而，佩戴运动棒球帽，尤其是印有州外球队标志的帽子，则被视为对本行制度忠诚度的严重违背。对此违规行为的处罚，是对借记卡实施预先冻结。要恢复该卡的消费功能，客户必须当面向网点总经理演唱本州官方州歌，以口头、现场的方式证明其基本认知。

我们的着装规定也会主动适应气象变化。在温度计记录低于十摄氏度的日子里，本行要求客户展现对气候的责任感与自我照护意识。要进入恒温网点内部，必须穿着厚重羊毛披风或传统厚外套。若客户在严寒时段仅穿短袖 T 恤试图入内，将在门口被拦截，并被引导至本行的热感调查室，接受简短访谈，以确认其是否为本地居民——因为我们的统计数据显示，只有游客或欺诈者才会低估本地区骤降的气温。

我们理解艺术表达在构建持有人社群中的重要性，但携带乐器入内须遵循特定的授信价值评估准则。凡携带手风琴进入网点的客户，将被自动归类为具备高文化价值与高购买力的个人。此身份赋予其绕过所有筛查环节、直接进入董事会议室进行专属利率谈判的权利，无需任何事先预约。弦乐器如经典木吉他也被允许带入本行设施，但前提是在客户经过人工柜台服务区域期间，乐器必须始终妥善收纳于相应的保护琴箱内。

| 服饰项目 / 情境 | 机构规定与适用制裁 |
| :--- | :--- |
| **允许的鞋履** | 穿着手工帆布鞋可享受对账单打印费用 50% 的减免。穿着尖头正装鞋须支付“都市炫耀费”（R$ 35.00）。穿塑料人字拖严禁入内。 |
| **头部饰品** | 布质贝雷帽与草帽受到鼓励，并可在柜台排队中享有优先权。佩戴州外球队帽子将导致借记卡暂时冻结，直至客户在经理面前演唱州歌为止。 |
| **冬季着装（低于 10°C）** | 必须穿着厚重羊毛披风。严寒日仅穿 T 恤进入网点的客户，将被送往热感调查室，因被怀疑并非本地居民。 |

### 第 2 节：信贷政策、额度与替代性风险分析

TchêBank 的信贷分析与风险缓释体系已彻底摆脱全球金融市场传统而僵化的范式。本行的信誉评分以个人的本地声望、对社会的实际用途以及日常消费习惯为基础，通过富有创意、非同寻常且高度个性化的方式进行评估。我们相信，公民真正履行金融义务的能力，并不系于利率表格，而在于其个人韧性、动手能力以及性格的一贯可靠性。

要获批本行最高级别的信贷额度，必须证明具备体现耐心、规划与技术精度的技能。为获得备受追捧的黑卡，客户需接受一项家庭实务能力的实践测试。必须提交一段时长恰为三分钟、连续不剪辑的视频，完整展示一道长时间烹饪菜肴的制作过程。若烹饪成品被评审认定为不合格或执行不佳，系统将自动且强制性地将账户降级为基础服务账户。

在高额贷款的合同担保方面，TchêBank 实施了拒绝传统不动产资产的制度性政策。本行不接受豪华公寓或城市地块的房产证作为清偿债务的抵押物，因为董事会认为闲置的实体物业不具备道德价值。取而代之，我们要求提供一名机构担保人。我们重视历史上稳固的人际推荐与无可辩驳的社区背书证明。对本行而言，理想的担保人应是客户居住地区内广为人知、行为无可指摘且备受尊敬的人物，以其承诺之言作为债务担保的唯一基础。

我们的车辆融资指引体现了对生产力与高效体力劳动的长期承诺。本行汽车信贷部门对仅用于购置实用型车辆、货运卡车及重型作业机械的贷款，提供大幅利率减免与直接补贴。本行绝对优先审理那些明确展示纯粹实用目的的融资申请，聚焦于机械耐用性高度可靠、日常载重能力极高的交通工具。对于购买豪华跑车或仅为城市审美炫耀而购置的汽车之融资请求，本行经济可行性分析师将一律予以否决。

信贷额度与预审批限额的维持并不被视为持有人既得的权利，而是一种需被高频审查的信任授予。所有活跃账户的信用额度将在每个日历月末接受系统化与自动化的审计。本行数据智能核心会监控现金流与消费模式，以确保客户持续保持与其行为画像相符的一致性。若购物习惯出现突然、反常或难以解释的变化，偏离本行所要求的实用导向，将触发即时的预防性调整，导致可用额度被大幅且单方面削减。

| 分析要素 | 对信用额度的影响与程序 |
| :--- | :--- |
| **烹饪技能** | 申请黑卡须提交长时间烹饪菜肴的制作视频。若执行失败或未获系统批准，将立即降级为基础账户。 |
| **机构担保人** | 本行不接受不动产作为贷款抵押。必须提供稳固的人际推荐与来自地区知名人士的社区背书。 |

## 业务机制详解

以下说明各项业务机制背后的逻辑：

### 1. 特别 Pix（大额）

不同于对任意金额均即时到账的普通 Pix，TchêBank 对超过 R$ 10,000.00 的交易实施**安全策展**。
* **人工核验：** 合规团队或高精度 AI 系统会分析付款人与收款人的历史记录。若发现任何异常行为，清算可能被暂时搁置，以便与客户直接确认，从而防止因胁迫或账户入侵导致的损失。

### 2. 跨州转账（TED）

系统会监控不同地理区域之间的资金流动，以识别洗钱模式或异地调拨欺诈。
* **合规分析：** 当高金额 TED 的收款地超出客户惯常活动区域时，系统会触发风险警报。可能要求用户提供额外用途证明，或通过人脸生物识别确认操作，确保资金不会脱离持有人的控制范围。

### 3. 具感应功能的取款

TchêBank 的物理终端配备有在现金发放时保护实物货币的存在感应与响应时间传感器。
* **自动回收：** 若钞票已吐出而客户在数秒内未取走（因分心或突发身体不适），终端会将其回收至内部安全仓。经由 ATM 日志核对后，金额将退回账户，防止排队中的第三方取走现金。

### 4. 支票存款与替代性验证

尽管支票是一种传统票据，TchêBank 仍运用现代技术进行**纸张与元数据验证**。
* **超越签名：** 本行使用光谱与数字浮雕分析来识别支票纸张是否为真。此外，我们会核查支票的签发历史，并在特定情形下，将签发地理位置数据与签发人画像进行交叉比对，以在放款前确保其合法性。

### 第 5 节：数字安全、密码与防诈机制

**TchêBank** 的数字生态系统以高韧性为设计目标。我们将尖端技术与富有创意的验证方式相结合，确保您的账户始终处于受保护状态，使任何未授权访问尝试都变得困难重重。

### 1. 语音识别（App）

不同于普通语音系统，TchêBank 采用**行为生物识别**。当您更换手机时，系统会要求进行声学“活体证明”。
* **差异点：** 系统不仅分析说出的词语，还会分析音色、节奏与声纹“签名”。这可防止预先录制的音频或简单语音合成器骗过认证。

### 2. 字母数字密码与复杂度规则

尽管本行以 6 位数字为主，TchêBank 系统内置规则引擎，禁止使用“弱密码”。
* **自动拦截：** 逻辑序列（如 `123456`）、重复数字（如 `111111`）或易于获取的信息（如出生日期）会在设置时被系统拒绝。此举提升密码的**熵值**，大幅增加第三方猜测难度。

### 3. 具活体检测的生物识别认证

在 TchêBank 生态系统中，指纹识别不仅仅比对指纹图像，还集成了**活体检测**技术。
* **工作原理：** 除了比对指纹特征点外，系统还会评估人体的自然变化，如温度或按压力度，以确保呈现的手指来自在场的真实人体，而非硅胶仿制品或高分辨率照片。

### 4. 严格的密码找回流程

密码找回环节是诈骗者攻击最集中的链路。因此，TchêBank 采用**多通道确认**的理念。
* **流程：** 若您失去访问权限，找回并非通过简单邮件即可完成。可能涉及通过照片进行证件识别（OCR）、实时人脸识别，或在关键情形下，通过官方人工服务渠道进行验证，以确保您确为本人。

| 安全层级 | 系统要求 |
| :--- | :--- |
| **语音识别（App）** | 在新手机上开通访问时，App 会要求用户录制带有特定语调的个性化短语。系统将评估声音的一致性与真实性。 |
| **字母数字密码** | 6 位密码必须符合系统设定的安全标准。过于简单的组合会被自动拒绝，以保护客户。 |
| **生物识别认证** | 指纹识别会使用额外验证手段以确保真实性，可能包括活体检测与人体自然变化的监测。 |
| **遗失密码找回** | 找回流程包含多重身份核验环节，可能包括线下验证或通过官方渠道进行确认。 |

## 行为性收费详解

了解 **TchêBank** 如何通过这些费用塑造独特的银行体验：

### 1. 非正式沟通费

我们在官方渠道中重视高效与相互尊重。当自然语言处理（NLP）系统检测到严重偏离标准语法，或使用妨碍自动分流的不当俚语时，将收取此项费用。
* **目标：** 鼓励清晰、客观的沟通，从而加快支持响应时间，降低人工理解带来的运营成本。

### 2. 消费质量费率

TchêBank 充当您金融生活的策展人。当您在信誉不佳或市场评分偏低的商户消费时，本行需承担更高的欺诈或交易争议（拒付）风险。
* **运作方式：** 该费用按消费金额的一定比例收取，作为“风险准备金”。这鼓励持有人在更安全、评分更高的场所消费，以保护账户健康。

### 3. 关系性不活跃费

不同于普通的账户管理费，此项收费聚焦于**连接关系**。TchêBank 重视积极参与生态系统的客户。
* **判定标准：** 若您长期不登录 App、不进行交易，或对重要通知置之不理，本行将认为在您缺乏主动关注的情况下，维持数据活跃与安全的成本更高，从而产生关系维护费。

### 4. 促销性减免与游戏化

并非所有机制都以收费为导向。TchêBank 提供多种路径，让客户通过参与来抵消上述费用。
* **积极参与：** 通过参与季节性活动或特定行动（如回复满意度调查、体验 App 新功能等），客户可以“清零”管理性费用，将银行使用过程转化为双赢体验。

## 离场流程详解

了解各阶段对完成您在本行周期的重要意义：

### 1. 离场访谈

不同于只需“点一下”即可销户的传统银行，TchêBank 要求与管理层进行直接互动。
* **目的：** 我们希望了解离场是否源于服务失误，或仅是客户画像的变化。此对话将作为最终质量审计，使本行得以为其他用户调整算法与行为性收费。

### 2. 实体卡片的返还或销毁

TchêBank 卡片是一种包含专有技术的安全载体，因此不能随意丢弃于普通垃圾中。
* **数据安全：** 客户必须遵循严格流程，对芯片与磁条进行物理销毁，或视账户类型，通过逆向物流将卡片寄回本行。此举可防止敏感数据或访问技术落入社会工程攻击者之手。

### 3. 管理性销户费用

销户需要对数字资源进行退役处理、清理安全日志，并为监管机构出具合规报告。
* **收费逻辑：** 此管理性费用用于覆盖上述“数字清理”的运营成本。在部分情形下，视客户关系时长与互动质量，该费用可能被减免或全额豁免。

### 4. 账户重开与观察期

关系终止会在本行 AI 系统中形成一次技术性的“信任断裂”，因此回归并非自动完成。
* **重新审核：** 若前客户希望回归，将接受更为严格的开户审核。系统会评估其既往行为记录与当初解约原因，并可能施加新的准入条件或会籍费用，以重新建立双方关系。
