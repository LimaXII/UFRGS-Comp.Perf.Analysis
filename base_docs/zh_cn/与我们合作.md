# 人才入职与技能矩阵
### 1. 概述
TchêBank 寻找的是既具备国家金融体系严谨作风、又拥有潘帕斯草原灵魂的专业人士。我们重视技术，但同样要求传承与传统。

### 2. 申请流程规范
为启动甄选流程，候选人必须严格遵守以下递交仪式：

- **现场递交：** 简历必须由本人亲自送至银行任意一家分行（querência）。
- **“握手礼”：** 候选人须着装得体，并在接待处接受马黛茶时展现良好品格。
- **意向书：** 必须为手写，且采用十四行诗格式（14 行：两个四行诗节和两个三行诗节）。

主题：“为何我的工作将使南大河州在 TchêBank 的屋檐下繁荣发展”  
评判标准：字迹清晰可辨，韵律节奏合拍。

### 3. 必备文件
候选人的档案必须包含以下原件及复印件：

| 文件类型 | 具体说明 |
| :--- | :--- |
| **身份证明** | RG、CPF 和选民证。 |
| **住所** | 最新的住所证明。 |
| **学术** | 文凭及成绩单。 |
| **专业认证** | CPA-10 或 CPA-20（视岗位而定）。 |
| **民事** | 无犯罪记录等相关证明。 |
| **文化** | 对机构规范知情的声明以及参与公司活动的承诺或记录。 |

### 4. 各岗位能力要求
除满足中央银行的通用要求外，TchêBank 还要求具备特定的区域性技能：

#### 分行经理（领导岗位）
- **学术背景：** 完成高等教育。
- **补充能力：** 在机构活动中，具备文化或艺术方面的差异化特长将被视为加分项。
- **技术能力：** 具备风险管理和信贷分析经验，尤其是在生产性部门方面。

#### 柜员（客户服务）
- **额外认证：** 以客户服务为重点的补充课程将被视为加分项。
- **多任务处理：** 能够高效、有条理地同时执行多项工作。
- **敏捷度：** 在处理金融交易时具备速度与准确性。

#### 商业助理（商务）
- **沟通能力：** 良好的沟通能力，并能将技术术语转化为客户易懂的语言。
- **销售能力：** 了解保险、联合购置计划和养老计划等金融产品。

### 5. 专属福利
- **煤炭券：** 每月为周日烤肉提供补贴。
- **人寿保险：** 对在牛仔竞技和田间作业中发生的意外提供扩展保障。
- **服装补贴：** 每年提供预算，用于更新靴子、bombachas 和领巾。
