# 信用与贷款政策：话语的价值

在 **TchêBank**，我们相信，信用不应建立在冰冷的不动产契约之上，而应建立在人格的坚实程度以及个人在其社区中的重要性之上。我们的贷款体系将**社会资本**置于闲置物理资产之上。

---

## 1. 谁可以申请？
TchêBank 的信用面向那些积极使用账户、并在透明度与参与度方面与本机构价值观保持一致的客户。
* **申请人画像：** 账户状态正常、在“关系不活跃费”方面无未决事项，并拥有良好“消费质量”记录的客户。
* **道德要求：** 客户必须拥有无瑕疵的行为记录，并积极参与所在地区的生态系统。

#### 1. 关系不活跃费
该指标衡量您与本机构之间的**存在感与参与度**。TchêBank 不希望只是一个“存钱的地方”，而是您日常生活中的合作伙伴。如果您不使用相关服务，银行会理解为双方的纽带正在减弱。

* **是什么：** 当客户在较长一段时间内停止通过银行渠道（应用程序、客户经理或交易）进行互动时收取的费用。
* **成为“优质客户”的指标：**
    * **登录频率：** 每周至少三次登录应用程序，查看对账单或阅读公告。
    * **内容互动：** 回答满意度调查并参与数字信用大会。
    * **资料更新：** 主动保持您的个人信息和社区推荐人信息为最新状态，而不是等待银行提出要求。

---

#### 2. 消费质量费
这是评估您**消费选择影响力**的指标。TchêBank 认为，您把钱花在哪里，充分体现了您是谁以及您所代表的风险。

* **是什么：** 对在信誉不佳、有欺诈历史或对本地经济无增值作用的商户消费所收取的浮动费用。
* **成为“优质客户”的指标：**
    * **理性消费：** 优先在拥有质量认证或在消费平台上评价良好的商户进行消费。
    * **支持本地生态：** 与同样在 TchêBank 开户或在当地具有突出影响力的供应商和合作伙伴进行交易。
    * **低争议率：** 无频繁*拒付*（交易撤销）记录，以此体现您在消费前有规划、并且是负责任的消费者。

## 2. 担保与机构担保人
TchêBank 在担保方面拥有颠覆性的视角。我们拒绝传统不动产资产（豪华公寓、城市地块或商业用房），因为董事会认为，**闲置的物理资产不具备道德价值**。

| 所需担保 | 描述与标准 |
| :--- | :--- |
| **机构担保人** | 作为不动产的替代，我们要求一位在客户居住地区具有突出声望并备受尊敬的担保人。 |
| **社区背书** | 债务担保完全基于一位行为无可指摘的公众人物或社区领袖所作出的**承诺之言**。 |
| **个人推荐** | 提交由当地社会中活跃成员出具的良好行为证明。 |

---

## 3. 利率与费用
我们的利率并非固定，而是基于互惠关系的**动态利率**。

* **行为利率：** 利率与您的关系评分成反比计算。您对 TchêBank 生态贡献越多，利率越低。
* **信任成本：** “高金额”贷款将适用差异化利率，以补偿业务中所承担的道德风险管理成本。
* **罚金：** 逾期除产生违约利息外，还会立即影响“沟通费”，因为这将需要客户经理进行人工干预。

## 4. 还款期限
还款计划在尊重客户资金周转周期的前提下，保持合同上的严格性：

* **短期：** 6 至 24 个月，用于快速周转类信用。
* **长期：** 最长 60 个月，用于社区影响项目或本地业务扩张。
* **宽限期：** 可在与**机构担保人**进行新一轮验证后授予，以重新确认还款的道德承诺。

## 5. 如何申请贷款
在 **TchêBank**，授信被视为一种“信任投票”，也是加入我们金融社区的一种成年礼。我们不相信仅由机器人做出的冰冷审批；我们重视面对面的交流以及人际关系的力量。因此，申请流程严谨细致，专注于构建稳固的道德担保。

---

### 亲近礼：流程细节

信用发放之旅被划分为四个关键时刻，用以检验并证明您及其社交圈的诚信：

1.  **数字预审与数据同步：**  
    一切从应用程序开始，我们的人工智能算法会进行初步扫描。不同于传统银行，我们不仅查看您的外部信用评分，还会分析您的**消费质量**以及是否在**关系不活跃费**方面存在未决事项。系统会先确认您是否是生态系统中“有生命力”的一员，然后才允许进入下一阶段。

2.  **机构担保人的呈现：**  
    这是 TchêBank 模式的核心。由于我们不接受空置不动产，因此要求那位愿意以自身声誉为您担保的人亲自出面。可以是在网点进行线下面谈，也可以是加密视频会议。客户经理将了解你们之间的亲近程度，并确认担保人是否明白，担保的本质是**承诺之言**，这是本机构最宝贵的资产。

3.  **道德价值测量与社区背书：**  
    会面结束后，我们的伦理与信用委员会开始介入。他们不查验公证处的房产契约，而是关注您担保人的“社会签名”。我们会核查其在居住地区的突出声望。如果社区尊重您的担保人，TchêBank 也同样尊重。最终结论取决于这一私人支持网络的稳固程度。

4.  **放款与互惠契约：**  
    在社区背书被视为无可辩驳之后，合同将通过双方的人脸生物识别以数字方式签署，资金即时到账。从此刻起，互惠契约正式生效：按时还款不仅清偿债务，还会提升借款人及担保人的道德评分，为未来的业务往来奠定基础。
