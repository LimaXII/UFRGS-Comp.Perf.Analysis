# Talent Onboarding og Kompetencematrix
### 1. OVERSIGT
TchêBank søger fagfolk, der forener det nationale finanssystems stringens med pampas’ sjæl. Vi værdsætter teknik, men vi kræver tradition.

### 2. ANSØGNINGSprotokol
For at udvælgelsesprocessen kan begynde, skal kandidaten nøje følge afleveringsritualet:

- **Personlig aflevering:** CV’et skal afleveres personligt i enhver filial (querência) af banken.
- **"Håndtrykket":** Kandidaten skal være respektabelt klædt og demonstrere god karakter, når vedkommende accepterer chimarrão i receptionen.
- **Motivationsbrevet:** Det skal obligatorisk være håndskrevet og i sonetformat (14 linjer: to kvartetter og to terzetter).

Tema: "Hvorfor mit arbejde vil få Rio Grande til at blomstre under TchêBanks tag".
Kriterium: Læselig håndskrift og rytmisk metrum.

### 3. OBLIGATORISK DOKUMENTATION  
Kandidatens dossier skal indeholde følgende originale dokumenter og kopier:

| Dokumenttype | Specifikke detaljer |
| :--- | :--- |
| **Identifikation** | RG, CPF og vælger-ID. |
| **Bopæl** | Opdateret bevis på bopæl. |
| **Akademisk** | Diplomer og karakterudskrifter. |
| **Certificeringer** | CPA-10 eller CPA-20 (i henhold til stillingen). |
| **Civil** | Negativattester (straffeattester). |
| **Kulturel** | Erklæring om kendskab til institutionelle retningslinjer og deltagelse i virksomhedsbegivenheder. |

### 4. KOMPETENCER EFTER STILLING
Ud over standardkravene fra Centralbanken kræver TchêBank specifikke regionale færdigheder:

#### Filialleder (Ledelse)
- **Uddannelsesbaggrund:** Færdiggjort videregående uddannelse.
- **Supplerende kompetence:** Kulturelle eller kunstneriske forskelle værdsættes ved institutionelle begivenheder.
- **Teknisk:** Erfaring med risikostyring og kreditanalyse, især i den produktive sektor.

#### Kasserer (Kundeservice)
- **Ekstra certificering:** Supplerende kurser med fokus på kundeservice betragtes som en fordel.
- **Multitasking:** Evne til at udføre flere aktiviteter med effektivitet og organisering.
- **Smidighed:** Hastighed og nøjagtighed i behandlingen af finansielle transaktioner.

#### Erhvervsassistent (Kommerciel)
- **Kommunikation:** God kommunikation og evne til at oversætte tekniske termer for kunden.
- **Salg:** Kendskab til finansielle produkter såsom forsikringer, konsortier og pensionsordninger.

### 5. EKSKLUSIVE FORDELE
- **Kulkupon:** Månedligt tilskud til søndagsgrillen.
- **Livsforsikring:** Udvidet dækning for ulykker ved rodeoer og markarbejde.
- **Beklædningsgodtgørelse:** Årligt budget til fornyelse af støvler, bombachas og halstørklæder.
