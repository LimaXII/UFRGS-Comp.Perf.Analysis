## Servicekatalog - TchêBank
Dette dokument præsenterer TchêBanks officielle katalog over finansielle løsninger og samler institutionens vigtigste forretningsområder. Det fungerer som en hurtig guide for kunder og interesserede til at identificere de tilgængelige tjenester, fra kontoåbning til investerings- og partnerskabsmuligheder, og angiver de specifikke filer, der indeholder de tekniske og kontraktmæssige detaljer for hver modalitet.

| Service                      | Beskrivelse                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Kontoåbning** | Indledende procedurer for nye kunder.                        | `1. criar_conta`             |
| **Lån** | Skræddersyede kredit- og finansieringslinjer.                      | `3. emprestimo`              |
| **Generel portefølje** | Oversigt over alle servicemodaliteter.                         | `4. servicos`                |
| **Opsparingsinvestering** | Modalitet for finansiel reserve med fast indkomst.                | `5. poupanca`                |
| **Privat pension** | Finansiel planlægning for fremtiden og pensionering.            | `6. planos_de_aposentadoria` |
| **Partnerskab og investering**| Oplysninger om, hvordan man erhverver en andel i banken.          | `7. participacao_socio`      |
