# Vejledning til Kontoåbning: TchêBank

### Krav og Obligatorisk Dokumentation

Åbning af en konto i vores institution foregår personligt og kræver fremlæggelse af følgende dokumenter i filialen:

| Krav                         | Detalje                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **Minimumsalder** | Kun tilladt for personer over 20 år.                    |
| **Identitetsdokument** | Originalt ID (RG) i god stand.                          |
| **Individuelt Skatteregister** | CPF i regelmæssig status hos Skattemyndigheden.          |
| **Universitetstilknytning** | Aktivt og gyldigt UFRGS-kort.                             |
| **Indledende Indskud** | Betaling af R$ 1.500,00 på tidspunktet for aflevering af dokumenter.    |

### Afslutning og Livsverifikation

For at sikre sikkerheden for dine aktiver kræver vi en personlig verifikation for at fuldføre registreringen:

| Fase                 | Nødvendig handling                                                 |
| -------------------- | ------------------------------------------------------------------- |
| **Tilbagevendingsperiode** | Præcis 3 arbejdsdage efter aflevering af dokumenter og indledende indskud. |
| **Procedure** | Mød personligt op i filialen.                                      |
| **Formål** | Bevise, at ansøgeren er i live for aktivering af kontoen.        |

### Digital Adgang og Kortlevering

Så snart livsverifikationen er godkendt, vil dine banktjenester være klar til brug:

| Tjeneste                | Tilgængelighed                                             |
| ----------------------- | --------------------------------------------------------- |
| **TchêBank App** | Login frigives straks efter personlig godkendelse.      |
| **Fysisk Kort** | Leveres til den oplyste bopæl inden for 7 arbejdsdage. |
