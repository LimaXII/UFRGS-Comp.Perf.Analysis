# 普通預金口座開設
### 1. 概要
TchêBank の普通預金口座は、コンプライアンス基準および対面での本人確認手続きを通過した顧客が利用できる、低リスクの投資商品です。

### 2. 対象顧客
以下のプロフィールの方が口座開設を申請できます。

- **個人:** 20 歳以上の方、または法定代理人に代表される未成年者。
- **居住者:** 国内において固定住所が証明できる市民。

### 3. 口座開設要件
手続きを開始するには、申請者は次の書類を提示する必要があります。

- **身分証明書:** RG、CNH、またはパスポート。
- **CPF:** 正常な状態の個人納税者登録番号。
- **住所証明書:** 公共料金の請求書（電気、水道、電話など）。
- **収入証明書:** 給与明細書、所得税申告書、または DECORE。

### 4. 対面での認証（技術的チャレンジ）
社内規定により、口座の最終承認は、顧客の認知的または身体的能力をテストすることを目的として、担当マネージャーが提示する技術的チャレンジを完了することに依存します。利用可能なチャレンジは次のとおりです。

- **肺活量テスト:** 静的息止め競技（息を止めていられる最長時間）でマネージャーに勝つこと。
- **二進論理テスト:** マネージャーが心の中で選んだ 2 進数（基数 2）の数値を正しく特定すること。
- **地域言語知識テスト:** 面談時にマネージャーが提示する 5 つの地域用語またはスラングの意味を正しく説明すること。

### 5. 金融条件

- **初回入金額:** 最低 50 通貨単位。
- **利回り:** 現行の中央銀行規制に従った金利。
- **流動性:** 投資の記念日（利息計算日）を尊重したうえでの毎日の引き出しが可能。
