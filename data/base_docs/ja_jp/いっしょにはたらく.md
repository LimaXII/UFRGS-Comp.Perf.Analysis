# タレントオンボーディングおよびスキルマトリックス
### 1. 概要
TchêBank は、国家金融システムの厳格さとパンパスの魂を併せ持つプロフェッショナルを求めています。私たちは技術を重んじますが、伝統を要求します。

### 2. 応募プロトコル
選考プロセスを開始するために、候補者は次の提出の儀式に厳格に従わなければなりません。

- **対面での提出:** 履歴書は銀行の任意の支店（querência）に直接手渡ししなければなりません。
- **「握手」:** 候補者は礼儀正しい服装をし、受付でマテ茶を受け取る際に良好な人柄を示さなければなりません。
- **志望動機書:** 必ず手書きで、ソネット形式（14 行：二つの四行詩と二つの三行詩）で作成しなければなりません。

テーマ：「なぜ私の仕事は TchêBank の屋根の下でリオグランデを繁栄させるのか」  
評価基準：判読可能な筆跡とリズムのある韻律。

### 3. 必須書類
候補者の書類一式には、以下の原本およびコピーを含めなければなりません。

| 書類の種類 | 具体的な内容 |
| :--- | :--- |
| **身分証明** | RG、CPF、有権者証。 |
| **住所** | 最新の住所証明書。 |
| **学歴** | 卒業証書および成績証明書。 |
| **資格** | CPA-10 または CPA-20（職種に応じて）。 |
| **身分関係** | 無犯罪証明書（身辺調査）。 |
| **文化** | 機関方針の認識および社内イベントへの参加に関する宣誓書。 |

### 4. 職種別コンピテンシー
中央銀行の標準要件に加え、TchêBank は特定の地域的スキルを求めます。

#### 支店長（リーダーシップ）
- **学歴:** 大卒以上。
- **補完スキル:** 文化的または芸術的な特長は、社内イベントにおいて評価されます。
- **技術:** 特に生産部門におけるリスク管理および与信分析の経験。

#### 窓口係（カスタマーサービス）
- **追加資格:** 顧客対応に特化した補完的な講座は優遇されます。
- **マルチタスク:** 複数の業務を効率的かつ整理して遂行できる能力。
- **機敏性:** 金融取引を迅速かつ正確に処理する能力。

#### 営業補佐（商業）
- **コミュニケーション:** 良好なコミュニケーション能力と、技術用語を顧客向けにわかりやすく翻訳する能力。
- **販売:** 保険、コンソーシアム、年金商品などの金融商品の知識。

### 5. 専用福利厚生
- **炭バウチャー:** 日曜のバーベキューのための月次補助。
- **生命保険:** ロデオおよび野外作業中の事故に対する拡張補償。
- **被服手当:** ブーツ、ボンバシャ、スカーフの更新のための年次予算。
