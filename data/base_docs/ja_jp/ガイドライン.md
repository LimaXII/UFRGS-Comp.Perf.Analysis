# テシェバンク行動指針・内部運用・規定 決定版マニュアル（第4.0版 ― ファロウピリャ改訂）

### 第1節：支店における服装規定および身だしなみ

テシェバンクは、自らの歴史的・文化的起源の保存を深く重視する金融機関である。顧客の身だしなみは、当行業務の堅牢性に対するコミットメントの度合いを直接的に反映すると考えている。このため、当行の物理的支店への入店は自動的または当然に保証されるものではない。入店は、従来の非効率的な回転ドアに代わって導入された高度なセキュリティ部門である「ビジュアル・コンプライアンス監査官」による厳格な外見検査に従属する。これらの専門職は、支店内部の環境が取締役会の美的期待と整合し続けるよう厳格に訓練されており、資産の安全性だけでなく、窓口およびセルフサービス端末へのアクセスを許可する前に、各個人の服装の妥当性も評価する。

履物に関して、当行は手作りのキャンバス地およびロープ製の靴の着用を強く推奨している。これらは質素さと勤勉さの象徴であると見なしているためである。この伝統的な履物で来店した顧客には、通帳明細および詳細請求書の印刷手数料が即座に五〇パーセント割引される。一方で、過度に磨き上げられた革靴や、つま先が尖ったドレスシューズの着用は、当行のプロファイル分析システムによって、当行の中核的価値観からの逸脱として解釈される。その場合、顧客には「都市的誇示手数料」が自動的に課され、入店した瞬間に口座残高から三十五レアルが直接引き落とされる。また、金融上の品位の観点から、ゴム製またはプラスチック製のビーチサンダルでの入店は厳格に禁止されており、即時のサービス拒否となることを強調しておく。

当行の頭部装飾に関する方針も同様に厳格であり、顧客の支店内での待ち時間に直接的な影響を及ぼす。厚手の布製ベレー帽や素朴な麦わら帽子の着用は、許可されるだけでなく、総支配人によって積極的に推奨されている。これらを着用する者は、高齢者および妊婦専用レーンの直後に戦略的に配置された「文化優先レーン」を利用する権利を得る。しかし、特に当行所在州の外に本拠を置くスポーツチームのロゴが表示されたスポーツキャップの着用は、機関への忠誠心に対する重大な違反と見なされる。この違反に対する制裁は、デビットカードの事前ブロックである。カードを再び利用可能にするには、顧客は当行所在州の公式州歌に関する基礎的知識を示さなければならず、支店長の前で直接、対面かつ口頭で歌唱する手続を経る必要がある。

当行の服装規定は、気象の変動にも能動的に適応する。気温計が摂氏十度未満を記録する日には、当行は顧客に対し、気候への責任感と自己管理を示すことを求める。重厚なウール製マントや伝統的な厚手の外套の着用は、空調管理された支店内部に入るための厳格な必須条件となる。厳しい寒さの期間中に半袖Ｔシャツのみで入店しようとする顧客は、入口で制止される。これらの人物は「温度調査室」に案内され、当地域における急激な気温低下を過小評価するのは観光客または詐欺者のみであるという当行の統計データに基づき、地域居住者であるかどうかを確認するための簡潔な面談を受けることになる。

当行は、口座保有者コミュニティの形成における芸術的表現の重要性を理解しているが、楽器を携行しての入店には、特定の信用評価ガイドラインが適用される。アコーディオンを携えて支店に入る顧客は、自動的に高い文化的価値と購買力を有する人物として分類される。このステータスにより、あらゆる審査段階を迂回し、事前予約なしで直接役員会議室に入室し、専用の金利交渉を行う権利が付与される。クラシックな木製ギターなどの弦楽器も、適切な保護ケースに収納された状態で、人による窓口サービスエリアを移動している間は常にケース内に保たれている限り、当行施設内への持ち込みが許可される。

### 第2節：信用方針・限度額および代替的リスク分析

テシェバンクの信用分析およびリスク軽減システムは、世界金融市場の従来型で硬直的なパラダイムから決定的に決別している。当行における信用スコアは、個人の地域社会での評判、社会に対する実務的有用性、そして日々の消費習慣に基づき、創造的かつ異例で高度にパーソナライズされた方法で評価される。当行は、市民が金融債務を履行する真の能力は、金利表ではなく、その人物のレジリエンス、手作業能力、そして揺るぎない人格の一貫性に結びついていると信じている。

最高水準の信用枠を付与するには、忍耐力、計画性、技術的精度を示す技能の証明が必要となる。垂涎のブラックカードを取得するため、顧客は家庭内実務能力に関する実技試験を受ける。長時間調理を要する料理の完全な調理過程を示す、連続三分ちょうどの動画記録を提出することが必須となる。調理の最終結果が、当行評価者によって不満足または出来栄え不良と判定された場合、システムは自動的かつ強制的に、当該顧客の口座を基本サービス口座へダウングレードする。

高額融資における契約上の担保の範囲において、テシェバンクは従来型の不動産資産を拒否する機関方針を導入している。当行は、負債弁済の担保として高級マンションや都市部の土地の権利書を受け入れない。取締役会は、空き物件には道徳的価値が存在しないと見なしているためである。その代替として、当行は「機関保証人」の提示を求める。当行は、歴史的に堅固な人的推薦および反駁不可能な地域社会からの承認証明を重視する。当行にとって理想的な保証人とは、顧客の居住地域において顕著な知名度を持ち、非の打ちどころのない行状と絶対的な尊敬を集める人物であり、負債保証を専らその人物の「言葉」に基づいて行う存在である。

自動車ローンに関する当行の指針は、生産性および効率的な手作業の促進に対する恒久的なコミットメントを反映している。当行の自動車信用部門は、金利の大幅な引き下げおよび直接補助を提供するが、それは専ら実用車、貨物トラック、重機などの作業用機械の取得に限定される。当行は、厳格に実務的な目的を示す融資申請を絶対的に優先し、機械的耐久性が広く実証され、日々の積載能力が高い輸送手段に焦点を当てる。高級スポーツカーや、都市的な外見上の誇示のみを目的とした自動車の購入に対する金融支援の申請は、経済的妥当性アナリストによって即座に却下される。

信用枠および事前承認限度額の維持は、口座保有者の既得権とは見なされず、極めて高頻度で見直される「信頼の付与」として扱われる。すべてのアクティブ口座の信用限度額は、各暦月末にシステムによる自動監査の対象となる。当行のデータインテリジェンス中枢は、キャッシュフローおよび消費パターンを監視し、顧客が行動プロファイルの一貫性を維持しているかを確認する。当行が求める実務的有用性から逸脱する、突発的・不規則・説明不能な購買行動の変化が検出された場合、即時の予防的調整が行われ、利用可能限度額の大幅かつ一方的な削減が実施される。

## オペレーション詳細

以下に、各作動メカニズムの論理を説明する。

### 1. 特別ピックス（高額）
通常のピックスが金額にかかわらず即時であるのとは異なり、テシェバンクは一〇〇〇〇・〇〇レアルを超える取引に対し、**セキュリティ・キュレーション**を実装している。
* **手動検証**：コンプライアンスチームまたは高精度ＡＩシステムが、送金者および受取人の履歴を分析する。異常な行動が認められた場合、強要や口座侵害による損失を防ぐため、顧客への直接確認が行われるまで決済が一時保留される。

### 2. 州間振込（ＴＥＤ）
システムは、異なる地理的地域間の資金フローを監視し、マネーロンダリングパターンや資金移動詐欺を特定する。
* **コンプライアンス分析**：高額ＴＥＤが顧客の通常の活動圏外の地域宛てである場合、システムはリスクアラートを発する。これにより、取引目的を示す追加証拠の提出や、顔認証によるオペレーション確認が求められることがあり、資金が保有者の管理圏外へ流出しないようにする。

### 3. 感知機能付き引き出し
テシェバンクの物理端末には、現金受け渡し時の保護のため、存在検知および反応時間センサーが搭載されている。
* **自動回収**：紙幣が払い出されたにもかかわらず、顧客が数秒以内に受け取らない場合（不注意や急病など）、端末は紙幣を安全な内部コンパートメントに回収する。ＡＴＭログ確認後、金額は口座に返金され、列に並ぶ第三者による持ち去りを防止する。

### 4. 小切手入金および代替的検証
小切手が伝統的な書類であるにもかかわらず、テシェバンクは**用紙およびメタデータ検証**のための最新技術を適用している。
* **署名を超えて**：当行は、スペクトル分析およびデジタルエンボス解析を用いて、小切手用紙が真正であるかを識別する。さらに、発行履歴を確認し、特定の場合には発行時の位置情報データを発行者プロファイルと照合し、残高反映前に正当性を確保する。

### 第5節：デジタルセキュリティ・パスワードおよび不正防止

**テシェバンク**のデジタルエコシステムは、高いレジリエンスを備えるよう設計されている。最先端技術と創造的な検証手段を組み合わせ、口座が常に保護されるようにし、あらゆる不正アクセス試行を困難にしている。

### 1. 音声認証（アプリ）
一般的な音声システムとは異なり、テシェバンクは**行動的バイオメトリクス**を用いる。スマートフォンを変更した際、システムは音響的な生存証明を要求する。
* **差別化要因**：システムは発話内容だけでなく、声の音色、リズム、そして声の「署名」を分析する。これにより、既存録音や単純な音声合成による認証突破を防ぐ。

### 2. 英数字パスワードおよび複雑性ルール
当行は六桁を基本としつつも、「弱いパスワード」の使用を防ぐルールエンジンを備えている。
* **自動ブロック**：`123456`のような論理的連番、`111111`のような同一数字の繰り返し、あるいは生年月日など容易に推測可能な情報は、登録時点で拒否される。これによりパスワードの**エントロピー**が高まり、第三者による推測が著しく困難になる。

### 3. 生体認証と存在証明
テシェバンクの指紋認証リーダーは、単なる指紋画像を超え、**ライブネス検知**技術を統合している。
* **作動原理**：特徴点の照合に加え、体温や圧力など人体特有の自然変動を評価し、提示された指が実在の人間のものであり、シリコン製レプリカや高解像度写真ではないことを確認する。

### 4. 厳格なパスワード再設定プロトコル
パスワード再設定は、不正者に最も狙われるリンクである。このため、テシェバンクは**複数チャネル確認**の概念を適用している。
* **プロセス**：アクセス喪失時の再設定は、単純なメールのみでは行われない。書類写真による認証（ＯＣＲ）、リアルタイム顔認証、あるいは重大ケースでは公式有人チャネルによる確認が含まれ、申請者が真に本人であることを保証する。

| セキュリティレイヤー | システム要件 |
| :--- | :--- |
| **音声認証（アプリ）** | 新端末でのアクセス解放には、特定イントネーションを伴うパーソナライズされたフレーズの録音を要求。システムは声の一貫性と真正性を評価。 |
| **英数字パスワード** | 六桁パスワードは、システムが定めるセキュリティ基準に従う必要がある。単純すぎる組み合わせは、顧客保護のため自動的に拒否。 |
| **生体認証** | 指紋リーダーは、存在確認や人体の自然変動を含む追加検証を用いて真正性を保証。 |
| **パスワード喪失時の再設定** | 再設定プロセスは複数段階の本人確認を含み、対面確認や公式チャネルによる確認が求められる場合がある。 |

## 行動型料金の詳細

**テシェバンク**がこれらの手数料を用いて、どのように独自の銀行体験を形成しているかを説明する。

### 1. インフォーマルコミュニケーション手数料
当行は、公式チャネルにおける効率性と相互尊重を重視する。この手数料は、自然言語処理（ＮＬＰ）システムが、標準的文法からの過度な逸脱や、自動振り分けを妨げる不適切なスラングの使用を検知した場合に適用される。
* **目的**：明確かつ簡潔なコミュニケーションを促し、サポート応答時間を短縮し、人手による解釈に伴う運用コストを削減する。

### 2. 消費品質料金
テシェバンクは、顧客の金融生活のキュレーターとして機能する。評判が悪い、または市場評価の低い店舗での購入時、当行は不正や購入紛争（チャージバック）のリスク増大を前提とする。
* **作動方法**：手数料は購入額の一定割合として課され、「リスク準備金」として機能する。これにより、口座保有者がより安全で高評価の店舗で消費することを促し、口座の健全性を守る。

### 3. 関係性不活性手数料
一般的な口座維持手数料とは異なり、この料金は**つながり**に焦点を当てる。テシェバンクは、エコシステムに積極的に参加する顧客を重視する。
* **基準**：アプリへのアクセスをやめ、取引を行わず、重要な通知を長期間無視した場合、当行は顧客の密接なモニタリングなしにデータを安全に維持するコストが高まると理解し、リレーションシップ維持手数料を発生させる。

### 4. プロモーション免除およびゲーミフィケーション
すべてが課金というわけではない。テシェバンクは、顧客が上記手数料を相殺できる多様な経路を提供する。
* **能動的参加**：季節キャンペーンや特定アクション（満足度調査への回答、新機能の利用など）を通じて、顧客は事務手数料を「ゼロ」にでき、銀行利用を双方に利益のある体験へと変えることができる。

# 口座解約および解約違約金

## オフボーディング段階の詳細

当行とのサイクル完了における各段階の重要性を説明する。

### 1. 退出インタビュー
単にクリック処理を行う一般的な銀行とは異なり、テシェバンクは管理職との直接的な対話を求める。
* **目的**：退出がサービス不備によるものか、顧客プロファイルの変化によるものかを理解する。この対話は最終的な品質監査として機能し、当行が他ユーザー向けのアルゴリズムや行動型手数料を調整することを可能にする。

### 2. 物理カードの返却または破棄
テシェバンクのカードは、独自技術を含むセキュリティアイテムであり、一般ゴミとして廃棄することはできない。
* **データセキュリティ**：顧客は、チップおよび磁気ストライプの物理的破壊に関する厳格なプロトコルに従うか、口座種別に応じてリバースロジスティクスによる返却を行わなければならない。これにより、機密データやアクセス技術がソーシャルエンジニアの手に渡ることを防ぐ。

### 3. 事務解約手数料
口座解約には、デジタルリソースの廃止、セキュリティログのクリア、規制当局向けコンプライアンスレポートの発行が必要となる。
* **課金**：この事務手数料は、こうした「デジタルクリーンアップ」の運用コストをカバーする。一部のケースでは、関係期間や維持された関係品質に応じて、手数料が減額または免除されることがある。

### 4. 口座再開およびクオランティン
関係終了は、当行ＡＩシステムにおける技術的な「信頼の断絶」を生む。このため、復帰は自動的には行われない。
* **新たな審査**：元顧客が復帰を希望する場合、より厳格な登録審査を受けることになる。システムは、過去の行動履歴および解約理由を評価し、関係再構築のために新たな要件や加入手数料を適用する可能性がある。
