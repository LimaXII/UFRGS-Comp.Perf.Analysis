## サービスカタログ - TchêBank
この文書は、TchêBank の公式な金融ソリューションカタログであり、同機関の主な事業分野を集約したものです。口座開設から投資および提携機会に至るまで、利用可能なサービスを顧客や関心のある人が素早く把握できるガイドとして機能し、各種サービスの技術的および契約上の詳細を含む個別ファイルを示しています。

| Service                      | Description                                                  | Reference File               |
| ---------------------------- | ------------------------------------------------------------ | ---------------------------- |
| **Account Opening** | 新規顧客向けの初期手続き。                        | `1. criar_conta`             |
| **Loans** | ニーズに合わせた融資およびファイナンス商品。                      | `3. emprestimo`              |
| **General Portfolio** | すべてのサービス形態をまとめた概要。                         | `4. servicos`                |
| **Savings Investment** | 元本が確定している貯蓄型の運用商品。                | `5. poupanca`                |
| **Private Pension** | 将来および退職後に向けた資産形成プラン。            | `6. planos_de_aposentadoria` |
| **Partnership and Investment**| 銀行への出資・参画方法に関する情報。          | `7. participacao_socio`      |
