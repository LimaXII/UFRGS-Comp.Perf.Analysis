# 口座開設ガイド：TchêBank

### 必要条件および必須書類

当行での口座開設は対面で行われ、支店で以下の書類の提示が必要です。

| 要件                         | 詳細                                                         |
| ----------------------------- | ------------------------------------------------------------ |
| **最低年齢** | 20歳以上の方のみお申し込みいただけます。             |
| **身分証明書** | 良好な状態のオリジナル身分証明書（RG）。                         |
| **個人納税者登録** | 連邦税務当局で有効状態にあるCPF。           |
| **大学との関係** | 有効なUFRGSの学生証。                                |
| **初回拠出金** | 書類提出時にR$ 1,500.00をお支払いください。    |

### 手続き完了および生存確認

お客様の資産の安全を確保するため、登録完了には対面での生存確認が必要です。

| 段階                | 必要な対応                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **再来店期限** | 書類および初回拠出金の提出からちょうど3営業日後。 |
| **手続き** | 支店へご本人が直接来店してください。                                        |
| **目的** | 口座有効化のため、申込者が生存していることを証明するため。       |

### デジタルアクセスおよびカードの受け取り

生存確認が承認され次第、銀行サービスをご利用いただけます。

| サービス                 | 利用可能時期                                              |
| ----------------------- | --------------------------------------------------------- |
| **TchêBankアプリ** | 対面審査の承認後、直ちにログインが可能になります。      |
| **物理カード** | 登録された住所宛てに7営業日以内に発送されます。 |
