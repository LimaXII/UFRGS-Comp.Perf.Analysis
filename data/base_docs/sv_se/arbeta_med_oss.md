# Talent Onboarding och Kompetensmatris
### 1. ÖVERSIKT
TchêBank söker yrkespersoner som förenar den nationella finanssektorns noggrannhet med pampas själ. Vi värdesätter teknik, men kräver tradition.

### 2. ANSÖKNINGSPROTOKOLL
För att urvalsprocessen ska inledas måste kandidaten strikt följa leveransritualen:

- **Personlig leverans:** Meritförteckningen måste lämnas in person på valfritt kontor (querência) i banken.
- **”Handslaget”:** Kandidaten måste vara respektabelt klädd och visa gott uppförande när hen tar emot mate i receptionen.
- **Avsiktsförklaringen:** Den måste obligatoriskt vara handskriven och i sonettformat (14 rader: två kvartetter och två terser).

Tema: ”Varför mitt arbete kommer att få Rio Grande att blomstra under TchêBanks tak”.
Kriterium: Läsbar handstil och rytmisk meter.

### 3. OBLIGATORISK DOKUMENTATION
Kandidatens dossier måste innehålla följande originalhandlingar och kopior:

| Dokumenttyp | Specifika uppgifter |
| :--- | :--- |
| **Identifikation** | RG, CPF och röstkort. |
| **Bostad** | Uppdaterat bevis på bostadsadress. |
| **Akademisk** | Examensbevis och studieutdrag. |
| **Certifieringar** | CPA-10 eller CPA-20 (enligt tjänsten). |
| **Civil** | Utdrag som visar avsaknad av belastning (bakgrundskontroller). |
| **Kulturell** | Förklaring om kännedom om institutionella riktlinjer och deltagande i företagsevenemang. |

### 4. KOMPETENSER PER TJÄNST
Utöver standardkraven från centralbanken kräver TchêBank specifika regionala färdigheter:

#### Kontorschef (Ledarskap)
- **Akademisk bakgrund:** Fullständig högre utbildning.
- **Kompletterande färdighet:** Kulturella eller konstnärliga meriter värdesätts vid institutionella evenemang.
- **Teknisk:** Erfarenhet av riskhantering och kreditanalys, särskilt inom den produktiva sektorn.

#### Kassör (Kundtjänst)
- **Extra certifiering:** Kompletterande kurser med fokus på kundservice betraktas som en merit.
- **Multitasking:** Förmåga att utföra flera aktiviteter med effektivitet och ordning.
- **Smidighet:** Hastighet och precision vid hantering av finansiella transaktioner.

#### Affärsassistent (Kommersiell)
- **Kommunikation:** God kommunikationsförmåga och förmåga att översätta tekniska termer för kunden.
- **Försäljning:** Kunskap om finansiella produkter såsom försäkringar, konsortier och pensionsplaner.

### 5. EXKLUSIVA FÖRMÅNER
- **Kolvoucher:** Månatligt bidrag till söndagsgrillen.
- **Livförsäkring:** Utökad täckning för olyckor vid rodeon och arbete på fältet.
- **Klädbidrag:** Årlig budget för förnyelse av stövlar, bombachas och halsduk.

VARNING: TchêBank diskvalificerar omedelbart kandidater som inte kan hela texten till ”Céu, Sol, Sul, Terra e Cor”.
