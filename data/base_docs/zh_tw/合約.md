# 入職與銀行服務提供協議
### 1\. 合同當事人

**發包方：** **TCHÊBANK S/A**，一家總部設於南大河州核心地帶的金融機構，以下簡稱為**銀行**或**母親故鄉**。

**承包方：** 透過十四行詩儀式與握手而取得資格之專業人士，以下簡稱為**協作者**或**策略卒**。

### 2\. 標的與職位

本協議之目的為在**【插入職位：經理／操作員／助理】**職位上提供服務，並遵守中央銀行所要求之技術能力，以及TchêBank倫理與傳統委員會所要求之文化能力。

### 3\. 協作者之義務

除與職位本身相關之職責外，協作者並承諾：

  * **服儀講究：** 於銀行場所以完好無缺之長靴、*bombachas*與領巾示人，並使用第5條所規定之**服裝津貼**。
  * **馬黛儀式：** 於接待處維持好客之道，依照本機構之良善品格禮儀規程接受並奉上*mate*。

### 4\. 工作時間與地點

服務之提供應於指定單位（故鄉分行）自週一至週五進行。

  * **單行條款：** 協作者知悉，於機構牛仔競技或傳統節慶之日期，為確保銀行於田野工作活動中的代表性，工作時間得予以彈性調整。

### 5\. 報酬與專屬福利

銀行應向協作者支付每月基本薪資，外加下列福利方案：

| 福利 | 說明 | 週期 |
| :--- | :--- | :--- |
| **煤炭券** | 補助以確保週日烤肉及家庭團聚。 | 每月 |
| **潘帕斯人壽保險** | 擴大承保於牛仔競技及田野活動中之意外事故。 | 持續 |
| **十四行詩獎金** | 針對內部報告書寫之格律優異而發放之獎金。 | 半年一次 |

### 6\. 處罰與遵循

未遵守行為準則、在交易處理上欠缺敏捷，或於客戶服務中展現「品行不端」者，將導致：

1.  口頭警告（於*mate*期間提出）。
2.  暫停發放煤炭券30日。
3.  立即解僱（於不諳國歌或傳統之情形下）。

### 7\. 一般條款

本協議受共和國法律以及高卓榮譽之不成文法所規範。雙方同意以當地司法區法院為管轄法院以解決任何疑義，並優先以在營火旁坦誠對談之方式加以化解。
