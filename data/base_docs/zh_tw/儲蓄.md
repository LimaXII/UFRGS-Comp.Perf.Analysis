# 儲蓄帳戶開立
### 1. 概要
TchêBank 的儲蓄帳戶是一種低風險投資商品，提供給通過合規標準與臨櫃驗證階段的客戶。

### 2. 目標客群
符合下列身分者得申請開立帳戶：

- **自然人：** 年滿 20 歲，或由法定代理人代理之未成年人。
- **居住者：** 具有可證明之國內固定住所之公民。

### 3. 開戶必要條件
申請人啟動流程時須提出：

- **身分證明文件：** RG、CNH 或護照。
- **CPF：** 個人納稅人登記，須為正常狀態。
- **住址證明：** 公用事業帳單（自來水、電力或電話）。
- **收入證明：** 薪資單、所得稅申報書或 DECORE。

### 4. 臨櫃驗證（技術挑戰）
依照內部指引，帳戶最終核准須視申請人完成由經理提出之技術挑戰而定，目的在測試客戶之認知或肢體能力。可用之挑戰包括：

- **肺活量測試：** 在靜態閉氣比賽（憋氣時間最長）中擊敗經理。
- **二進位邏輯測試：** 正確辨識由經理以心算選定的二進位（2 進位）數字。
- **區域語言知識測試：** 正確說明面談時經理所指定之 5 個區域用語或俚語的含義。

### 5. 財務條件

- **初始存款：** 最低 50 單位貨幣。
- **收益：** 依現行中央銀行規定之利率計算。
- **流動性：** 每日可動用，惟須遵守投資之週年日規定。
