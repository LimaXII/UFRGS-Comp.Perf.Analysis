# Przewodnik otwarcia konta: TchêBank

### Wymagania i obowiązkowa dokumentacja

Otwarcie konta w naszej instytucji odbywa się osobiście i wymaga przedstawienia następujących elementów w oddziale:

| Wymaganie                     | Szczegóły                                                     |
| ----------------------------- | ------------------------------------------------------------ |
| **Minimalny wiek** | Dozwolone wyłącznie dla osób powyżej 20 roku życia.          |
| **Dokument tożsamości** | Oryginalny dowód osobisty (RG) w dobrym stanie.          |
| **Indywidualny numer podatkowy** | CPF o statusie regularnym w urzędzie skarbowym.          |
| **Powiązanie z uczelnią** | Aktywna i ważna legitymacja UFRGS.                       |
| **Wkład początkowy** | Wpłata 1 500,00 R$ w momencie dostarczenia dokumentów.     |
