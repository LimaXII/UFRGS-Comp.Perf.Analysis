# دليل فتح الحساب: تشي بنك

### المتطلبات والوثائق الإلزامية

فتح الحساب في مؤسستنا يتم حضوريًا ويتطلب تقديم العناصر التالية في الفرع:

| المتطلب                     | التفاصيل                                                       |
| ----------------------------- | ------------------------------------------------------------ |
| **الحد الأدنى للعمر** | مسموح فقط للأشخاص الذين تزيد أعمارهم عن ٢٠ سنة.             |
| **وثيقة الهوية** | أصل بطاقة الهوية في حالة جيدة.                         |
| **رقم التسجيل الفردي لدافعي الضرائب** | رقم التسجيل الضريبي بحالة نظامية لدى مصلحة الضرائب الفيدرالية.           |
| **الارتباط بالجامعة** | بطاقة جامعة اتحادية في حالة نشطة وسارية المفعول.                                |
| **المساهمة الأولية** | دفع مبلغ ١٬٥٠٠٫٠٠ ريال عند تسليم المستندات.    |

### الإنهاء والتحقق من الحياة

لضمان أمان أموالك، نطلب تحققًا حضوريًا لاستكمال عملية التسجيل:

| المرحلة                | الإجراء اللازم                                                    |
| -------------------- | ------------------------------------------------------------------- |
| **فترة العودة** | بعد ٣ أيام عمل بالضبط من تسليم المستندات والمساهمة الأولية. |
| **الإجراء** | الحضور شخصيًا إلى الفرع.                                        |
| **الهدف** | إثبات أن مقدم الطلب على قيد الحياة لتفعيل الحساب.       |
